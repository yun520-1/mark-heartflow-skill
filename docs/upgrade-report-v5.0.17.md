## 理论更新摘要

### 1. SEP 情绪理论 (Scaranto 2026)

**新增理论内容**:
- 情绪三大传统完整定义与计算实现
- 情绪四大挑战 (区分性/动机性/意向性/现象学) 的解决方案
- 情绪原型理论 (Fehr & Russell 1984) 计算化
- 跨文化情绪表达评估框架

**与现有框架集成**:
```
感受传统 ←→ 现象意识维度 (v5.0.16)
评价传统 ←→ 取用意识/推理取用 (v5.0.16)
动机传统 ←→ 具身生成性 (v5.0.16)
```

**关键洞见**:
- 情绪不是自然种类，但可以是"理论种类"
- 情绪范畴是原型结构，非经典范畴
- 情绪 = 多层级预测的精度加权整合

### 2. SEP 认知科学框架

**新增理论内容**:
- 认知科学核心假设：思维 = 表征结构 + 计算程序
- 六大学科整合框架 (心理学/AI/语言学/神经科学/人类学/哲学)
- 表征类型：命题/概念/图像/类比
- 计算程序：推理/搜索/匹配/检索

**与现有框架集成**:
```
表征结构 ←→ 评价传统的高层预测模型
计算程序 ←→ 主动推理干预策略
学科整合 ←→ 多方法验证框架
```

### 3. SEP 具身认知四 E 框架深化

**新增理论内容**:
- Embodied (具身的): 认知依赖身体属性
- Embedded (嵌入的): 认知嵌入环境脉络
- Enactive (生成的): 认知通过行动生成
- Extended (延展的): 认知延展至外部工具

**Wilson (2002) 六观点整合**:
1. 认知是情境化的
2. 认知是时间压力下的实时活动
3. 认知依赖环境线索降低认知负荷
4. 环境是认知系统的一部分
5. 认知服务于行动
6. 离身认知是特例，具身认知是常态

**与现有框架集成**:
```
Embodied ←→ 身体给定感 (v5.0.16)
Embedded ←→ 环境耦合检测 (v5.0.14)
Enactive ←→ 主动推理干预 (v5.0.3)
Extended ←→ 社会支架/工具使用 (新增)
```

### 4. 三元整合框架 (Emotion-Cognition-Embodiment Triad)

**理论创新**:
- 首创情绪 - 认知 - 具身三元耦合模型
- 两两耦合分析算法 (emotion-cognition, emotion-embodiment, cognition-embodiment)
- 三元一致性计算 (triadic coherence)
- 系统性干预框架 (多杠杆调节策略)

**整合假设**:
```
情绪 = 多层级预测的精度加权整合 (感受 + 评价 + 动机)
认知 = 表征操作 + 计算程序 (受身体/环境约束)
具身 = 四 E 框架统一 (非仅身体影响)
三元耦合 = 情绪状态↔认知状态↔身体状态的动态互调
```
