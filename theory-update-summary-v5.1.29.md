## Core Updates | 核心更新

### 1. Husserlian Time-Triplet Integration | 胡塞尔时间三元组整合

**English:**

The system now models emotional experience through Husserl's three-fold temporal structure:

- **Retention** (Primary Memory): Just-past phase held in consciousness
  - Emotional residue tracking
  - Affective momentum calculation
  - Retention decay modeling (exponential decay, τ = 2-5 seconds)

- **Primal Impression** (Now-Point): Immediate present experience
  - Emotional intensity peak detection
  - Phenomenological thickness assessment
  - Present-moment awareness calibration

- **Protention** (Anticipation): Forward-looking expectation
  - Emotional trajectory prediction
  - Affective forecasting (Wilson & Gilbert model)
  - Expectation-violation detection

**Integration Formula:**
```
EmotionalExperience(t) = α·Retention(t-Δt) + β·PrimalImpression(t) + γ·Protention(t+Δt)
where α + β + γ = 1.0 (typical: α=0.3, β=0.4, γ=0.3)
```

**中文:**

系统现在通过胡塞尔的三重时间结构建模情绪体验：

- **滞留** (初级记忆)：意识中保持的刚过去阶段
  - 情绪残留追踪
  - 情感动量计算
  - 滞留衰减建模 (指数衰减，τ = 2-5 秒)

- **原印象** (现在点)：即时当下体验
  - 情绪强度峰值检测
  - 现象学厚度评估
  - 当下觉察校准

- **前摄** (预期)：向前看的期望
  - 情绪轨迹预测
  - 情感预测 (Wilson & Gilbert 模型)
  - 期望违背检测

**整合公式:**
```
情绪体验 (t) = α·滞留 (t-Δt) + β·原印象 (t) + γ·前摄 (t+Δt)
其中 α + β + γ = 1.0 (典型值：α=0.3, β=0.4, γ=0.3)
```

### 2. Jamesian Specious Present Enhancement | 詹姆斯显似现在增强

**English:**

William James characterized the "specious present" as the short duration of which we are immediately aware—a temporal window of approximately 0.5 to 3.0 seconds. The system now:

- **Temporal Window Calibration**: Adjusts specious present duration based on:
  - Emotional arousal (high arousal → shorter window)
  - Attention focus (focused → narrower window)
  - Flow state (optimal → 1.5-2.0 seconds)

- **Experience Thickness Assessment**: Evaluates the "density" of emotional experience within the specious present:
  - Thin experience: Single emotion, low complexity
  - Medium experience: 2-3 emotions, moderate complexity
  - Thick experience: Multiple emotions, high complexity, rich phenomenology

- **Temporal Granularity Mapping**: Creates fine-grained temporal profiles of emotional episodes

**中文:**

威廉·詹姆斯将"显似现在"描述为我们直接觉察的短暂持续时间——一个约 0.5 至 3.0 秒的时间窗口。系统现在：

- **时间窗口校准**：根据以下因素调整显似现在持续时间：
  - 情绪唤醒度 (高唤醒 → 更短窗口)
  - 注意力聚焦 (聚焦 → 更窄窗口)
  - 心流状态 (最优 → 1.5-2.0 秒)

- **体验厚度评估**：评估显似现在内情绪体验的"密度"：
  - 薄体验：单一情绪，低复杂度
  - 中等体验：2-3 种情绪，中等复杂度
  - 厚体验：多种情绪，高复杂度，丰富现象学

- **时间粒度映射**：创建情绪 episodes 的细粒度时间画像

### 4. Emotion-Time Cross-Analysis | 情绪 - 时间交叉分析

**English:**

The system now performs cross-analysis between emotional states and temporal experience:

- **Temporal Emotion Profiling**: Maps emotions to their characteristic temporal signatures:
  - **Fear**: Short-duration, high-intensity, sharp onset/offset
  - **Anger**: Medium-duration, escalating intensity, slow decay
  - **Sadness**: Long-duration, sustained low-medium intensity, gradual decay
  - **Joy**: Variable duration, can be sustained, often has afterglow
  - **Anxiety**: Extended duration, fluctuating intensity, anticipatory

- **Emotion Lifecycle Prediction**: Predicts emotional episode duration and trajectory:
  - Based on eliciting event type
  - Current temporal depth
  - Retention-protention balance

- **Time-Based Intervention Selection**: Selects interventions based on temporal characteristics:
  - Short-duration emotions → Acceptance/riding the wave
  - Long-duration emotions → Cognitive reappraisal, meaning reconstruction
  - Anticipatory emotions → Protention recalibration

**中文:**

系统现在执行情绪状态与时间体验的交叉分析：

- **时间情绪画像**：将情绪映射到其特征时间签名：
  - **恐惧**: 短持续时间，高强度，急剧 onset/offset
  - **愤怒**: 中等持续时间，升级强度，缓慢衰减
  - **悲伤**: 长持续时间，持续低 - 中强度，逐渐衰减
  - **喜悦**: 可变持续时间，可维持，常有余晖
  - **焦虑**: 延长持续时间，波动强度，预期性

- **情绪生命周期预测**：预测情绪 episode 的持续时间和轨迹：
  - 基于引发事件类型
  - 当前时间深度
  - 滞留 - 前摄平衡

- **基于时间的干预选择**：根据时间特征选择干预：
  - 短持续时间情绪 → 接受/乘风破浪
  - 长持续时间情绪 → 认知重评，意义重构
  - 预期性情绪 → 前摄重新校准

## New Assessment Tools | 新增评估工具

### 1. Temporal Depth Scale (TDS) | 时间深度量表

**English:**
A 10-item scale measuring subjective experience of temporal depth:
- Past integration (3 items)
- Present awareness (4 items)
- Future orientation (3 items)
- Score range: 0.0-1.0 (higher = richer temporal depth)

**中文:**
10 项量表测量主观时间深度体验：
- 过去整合 (3 项)
- 当下觉察 (4 项)
- 未来导向 (3 项)
- 评分范围：0.0-1.0 (越高 = 越丰富的时间深度)

### 2. Specious Present Duration Test (SPDT) | 显似现在持续时间测试

**English:**
Calibrates individual's specious present window through:
- Temporal bisection tasks
- Emotional episode duration estimation
- Flow state temporal markers
- Output: Optimal window duration (0.5-3.0 seconds)

**中文:**
通过以下方式校准个体的显似现在窗口：
- 时间二分任务
- 情绪 episode 持续时间估计
- 心流状态时间标记
- 输出：最优窗口持续时间 (0.5-3.0 秒)

### 3. Emotion Temporal Profile (ETP) | 情绪时间画像

**English:**
Creates detailed temporal signatures for each emotion:
- Onset dynamics (sharp/gradual)
- Peak characteristics (single/multiple peaks)
- Decay pattern (exponential/linear/sustained)
- Afterglow duration
- Typical duration range

**中文:**
为每种情绪创建详细的时间签名：
- 起始动力学 (急剧/逐渐)
- 峰值特征 (单峰/多峰)
- 衰减模式 (指数/线性/持续)
- 余晖持续时间
- 典型持续时间范围
