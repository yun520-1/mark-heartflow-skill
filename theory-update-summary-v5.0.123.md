### 1.2 新增理论条目 (4 个)

#### 1.2.1 预测加工情绪理论 (Predictive Processing of Emotion)
- **核心概念**: predictive coding, prediction error, active inference, free energy principle
- **理论来源**: 
  - Clark (2013). *Whatever Next? Predictive Brains, Situated Agents, and the Future of Cognitive Science*
  - Friston (2010). *The Free-Energy Principle: A Unified Brain Theory*
  - Seth (2013). *Interoceptive Inference, Emotion, and the Embodied Self*
- **核心主张**: 情绪是大脑通过预测加工生成的最佳假设，用于解释内感受信号
- **集成位置**: `predictive-emotion-v5.0.1`
- **进化意义**: 将情绪理论从"反应式"转向"生成式"范式

#### 1.2.2 具身预测情绪模型 (Embodied Predictive Emotion)
- **核心概念**: embodied prediction, interoceptive inference, allostasis, body-budget
- **理论来源**: 
  - Barrett (2017). *How Emotions Are Made: The Secret Life of the Brain*
  - Barrett & Simmons (2015). *Interoceptive Predictions in the Brain*
  - Seth & Critchley (2013). *Extending Predictive Processing to the Body*
- **核心主张**: 情绪预测本质上是具身的，基于身体预算 (body-budget) 的调节需求
- **集成位置**: `embodied-predictive-emotion-v5.0.2`
- **进化意义**: 整合具身认知与预测加工，形成统一的情绪生成模型

#### 1.2.3 集体意向性的规范维度 (Normative Dimension of Collective Intentionality)
- **核心概念**: joint commitment, obligation, entitlement, normative expectation
- **理论来源**: 
  - Gilbert (1990). *Walking Together: A Paradigmatic Social Phenomenon*
  - Gilbert (2006). *A Theory of Political Obligation*
  - Tuomela (2007). *The Philosophy of Sociality*
- **核心主张**: 集体意向性涉及联合承诺，产生相互的义务和资格
- **集成位置**: `collective-normative-v5.0.1`
- **进化意义**: 从认知维度扩展到规范维度，增强集体情绪的解释力

#### 1.2.4 集体情绪现象学 (Phenomenology of Collective Emotion)
- **核心概念**: shared feeling, emotional contagion, group emotion, we-experience
- **理论来源**: 
  - Scheler (1954 [1912]). *The Nature of Sympathy*
  - Walther (1923). *Zur Ontologie der sozialen Gemeinschaften*
  - Salmela & Nagatsu (2016). *How Does It Really Feel to Act Together?*
- **核心主张**: 集体情绪不是个体情绪的聚合，而是不可还原的共享体验
- **集成位置**: `collective-emotion-phenomenology-v5.0.2`
- **进化意义**: 深化集体情绪的现象学理解，区分真实共享与情绪感染

#### 3.1.1 预测误差计算模块
```javascript
function calculatePredictionError(predictedState, actualState, precision) {
  // 预测误差 = (实际值 - 预测值) × 精度权重
  const error = actualState - predictedState;
  const weightedError = error * precision;
  
  return {
    rawError: error,
    weightedError: weightedError,
    precision: precision,
    surpriseLevel: Math.abs(weightedError) // 惊奇度
  };
}
```

## 四、干预策略增强

### 4.1 新增干预方法 (7 种)

#### 4.1.1 预测误差最小化训练
- **目标**: 提高情绪预测准确性，减少预测误差
- **技术**: 
  1. 情绪预测日志 (预测即将发生的情绪)
  2. 实际体验记录 (记录实际发生的情绪)
  3. 误差分析 (比较预测与实际)
  4. 模型更新 (调整情绪概念)
- **理论来源**: Clark (2013), Barrett (2017)

#### 4.1.2 身体预算调节练习
- **目标**: 优化身体预算管理，预防情绪失调
- **技术**:
  1. 身体预算扫描 (评估当前能量储备)
  2. 支出预测 (预测未来能量需求)
  3. 预算平衡策略 (休息、营养、社交)
  4. 预防性调节 (在预算赤字前干预)
- **理论来源**: Barrett (2017), Sterling (2004)

#### 4.1.3 主动推理行动促进
- **目标**: 通过改变身体状态来调节情绪
- **技术**:
  1. 识别预测误差来源
  2. 选择行动策略 (运动、呼吸、姿势)
  3. 执行身体行动
  4. 监测误差减少
- **理论来源**: Friston (2010), Seth (2013)

#### 4.1.4 集体情绪同步练习
- **目标**: 增强群体情绪协调和凝聚力
- **技术**:
  1. 同步呼吸练习
  2. 联合注意力训练
  3. 情绪镜像练习
  4. 集体意向性声明
- **理论来源**: Scheler (1954), Salmela & Nagatsu (2016)

#### 4.1.5 联合承诺强化
- **目标**: 增强群体成员间的承诺和义务感
- **技术**:
  1. 共同目标明确化
  2. 相互期望表达
  3. 承诺公开声明
  4. 规范性反馈
- **理论来源**: Gilbert (1990, 2006)

#### 4.1.6 情绪原型重构
- **目标**: 调整情绪范畴原型，提高情绪粒度
- **技术**:
  1. 情绪原型探索 (识别当前原型)
  2. 原型边界扩展 (学习新的情绪概念)
  3. 典型性重评 (重新评估情绪体验)
  4. 概念整合 (整合新旧原型)
- **理论来源**: Fehr & Russell (1984), Barrett (2017)

#### 4.1.7 具身预测觉察练习
- **目标**: 提高对身体预测过程的觉察
- **技术**:
  1. 身体预测扫描 (识别身体预期)
  2. 预测来源分析 (情境/身体/先验)
  3. 预测灵活性训练 (尝试不同预测)
  4. 预测 - 体验对比
- **理论来源**: Seth (2013), Barrett & Simmons (2015)
