### 集体意向性 We-Intention 检测器
```
检测流程:
1. 语言标记分析
   - "我们" vs "我"使用频率
   - 共同动词 (一起做/一起想/一起感受)
   - 承诺性语言 (答应/保证/约定)

2. 互惠性评估
   - 相互期望检测
   - 信任水平评估
   - 联合承诺识别

3. 集体情绪识别
   - Scheler 共享体验四层:
     a) 同感 (Füreinander)
     b) 同向 (Miteinander)
     c) 共同 (Gemeinsam)
     d) 融合 (Ineinander)

4. We-Intention 置信度
   - 个体意图聚合 → 低置信度 (0.3-0.5)
   - 联合承诺 + 互惠期望 → 中置信度 (0.5-0.7)
   - 完整四层共享体验 → 高置信度 (0.7-0.9)
```

```prolog
% 自我意识谓词

% 意识谓词
conscious_state(Agent, Content, Mode).

% 元认知谓词
metacognitive_judgment(Agent, Type, Value).
confidence_calibration(Agent, Domain, Accuracy).

% 集体意向性谓词
we_intention(Group, Action, Commitment_Level).
joint_commitment(Agent1, Agent2, Action).
collective_emotion(Group, Emotion, Sharing_Level).
```

### 推理规则库 (部分)
```prolog
% 情绪识别规则 (评价传统)
recognize_emotion(Agent, Fear) :-
    appraisal(Agent, Situation, dangerous, high),
    appraisal(Agent, Situation, controllable, low),
    physiological_response(Agent, [heart_rate↑, breathing↑]).

% 元认知干预规则
generate_metacognitive_intervention(Agent, Strategy) :-
    metacognitive_judgment(Agent, JOL, low),
    confidence_calibration(Agent, Domain, Accuracy),
    Accuracy < 0.7,
    Strategy = elaborative_rehearsal.

% 集体意向性检测规则
detect_we_intention(Group, Action, Confidence) :-
    language_markers(Group, we_pronouns, Freq),
    Freq > 0.6,
    joint_commitment_exists(Group, Action),
    mutual_trust(Group, Level),
    Level > 0.5,
    Confidence is (Freq + Level) / 2.

% 具身认知干预规则
generate_embodied_intervention(Agent, Strategy) :-
    body_state(Agent, Tension, high),
    emotion(Agent, Type, Intensity, _),
    Intensity > 0.7,
    Strategy = body_scan_relaxation.
```
