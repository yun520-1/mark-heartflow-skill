**来源**: Stanford Encyclopedia of Philosophy - Consciousness  
**整合维度**:
- 现象意识 (Phenomenal Consciousness) → 情绪感受质基础

3. 取用意识支持情绪成分的可推理取用

5. 监控意识实现情绪原型典型性校准

### 3.2 情绪原型典型性计算 v2.1

```javascript
// 情绪原型典型性评分 v2.1
function calculateEmotionTypicality(emotionCategory, instance) {
  const prototype = emotionPrototypes[emotionCategory];
  
  // 五成分匹配 (SEP Emotion §1)
  const componentMatches = {
    evaluative: similarity(instance.appraisal, prototype.appraisal),
    physiological: similarity(instance.arousal, prototype.arousal),
    phenomenological: similarity(instance.feeling, prototype.feeling),
    expressive: similarity(instance.expression, prototype.expression),
    behavioral: similarity(instance.actionTendency, prototype.actionTendency)
  };
  
  // 加权典型性 (Fehr & Russell 1984)
  const typicality = 
    0.25 * componentMatches.evaluative +
    0.20 * componentMatches.physiological +
    0.25 * componentMatches.phenomenological +
    0.15 * componentMatches.expressive +
    0.15 * componentMatches.behavioral;
  
  // 置信度评估
  const confidence = calculatePrototypeConfidence(instance, prototype);
  
  return {
    typicality,
    confidence,
    componentProfile: componentMatches
  };
}
```

## 四、新增干预策略

### 4.1 现象学还原干预 v2.0

**目标**: 增强现象意识深度，提升情绪感受质觉察

**步骤**:
1. **悬置判断 (Epoché)**: 暂停对情绪的评价性反应
2. **回到事物本身**: 直接描述情绪体验的感受质特征
3. **本质直观**: 识别情绪体验的核心结构
4. **第一人称确认**: 确认体验的第一人称给定性

**适用场景**:
- 情绪模糊或混乱时
- 需要深化情绪觉察时
- 情绪调节困难时

### 4.2 自我意识双层校准 v1.0

**目标**: 平衡前反思与反思自我意识，提升自我 - 情绪整合

**步骤**:
1. **前反思觉察**: 注意情绪体验的直接给定性
2. **反思理解**: 对情绪进行概念化和因果分析
3. **双层对话**: 在前反思体验与反思理解之间建立对话
4. **整合确认**: 确认双层自我意识的连贯性

**适用场景**:
- 自我 - 情绪疏离时
- 情绪理解困难时
- 需要提升自我觉察时

### 4.3 集体意识增强干预 v1.0 (新增)

**目标**: 增强集体意向性意识，提升共享情绪体验

**步骤**:
1. **We-Intention 识别**: 识别集体意图的存在
2. **联合承诺觉察**: 觉察集体承诺的约束力
3. **共享体验确认**: 确认情绪体验的共享性
4. **相互责任承担**: 承担集体情绪中的相互责任

**适用场景**:
- 团队情绪协调时
- 集体创伤处理时
- 群体认同强化时
