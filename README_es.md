## 🔐 Auditoría de Seguridad v10.4.0

### ✅ Problemas Corregidos
- Límites de longitud de entrada (máx. 50,000 caracteres)
- Prevención de inyección HTML/script
- Detección de crisis de tiempo constante
- Iteración limitada (previene ReDoS)
- Operaciones de motor thread-safe
- Sin datos sensibles en mensajes de error
- Verificación de límites de enteros type-safe
- Límites de recursos en todas las colecciones

## 📦 Instalación Universal

```bash
git clone https://github.com/yun520-1/mark-heartflow-skill.git
cd mark-heartflow-skill
./install.sh
```

### Uso Básico

```python
from heartflow import HeartFlow, process_input

result = process_input("Ayudar a otros me hace feliz")
print(result.decision)
print(result.emotion_analysis['primary'])
```
