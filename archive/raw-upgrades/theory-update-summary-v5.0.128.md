## 一、新增理论模块

### 1.1 主动推理情感框架 (Active Inference Framework for Emotion)

**理论来源**: 
- Friston, K. & Barrett, L. F. (2025). Emotion as active inference.
- Clark, A. (2024). Predictive processing and the self.
- SEP: Predictive Processing (2024)

**核心整合点**:
1. **情绪作为内感受性预测**
   - 情绪范畴是大脑对身体状态的最优预测假设
   - 情绪波动 = 预测误差的调节过程
   - 主动推理：通过行动最小化长期预测误差

2. **预测误差追踪模块**
   - 预期情绪 vs 实际体验的误差计算
   - 误差最小化策略生成
   - 个性化内感受性预测模型

3. **临床应用重新概念化**
   - 抑郁症 = 内感受性预测偏差的累积
   - 焦虑症 = 预测不确定性过高
   - 干预目标：校准预测模型而非消除症状

### 1.2 4E 具身认知整合框架 (4E Cognition Integrated Framework)

**理论来源**:
- SEP: Embodied Cognition (Winter 2024)
- Merleau-Ponty 身体现象学重新评估
- 具身预测加工 (Embodied Predictive Processing)

**核心整合点**:
1. **4E 框架完整集成**
   - Embodied (具身的): 身体状态纳入情感分析
   - Embedded (嵌入的): 环境情境建模
   - Extended (延展的): 外部工具作为认知延伸
   - Enactive (生成的): 情绪通过行动生成

2. **身体信号整合**
   - 姿势、运动、生理信号的情感意义
   - 具身干预：身体动作调节情绪
   - 身体 - 情绪耦合追踪

3. **环境情境建模**
   - 物理环境对情感状态的影响
   - 社会情境的情感调节作用
   - 情境 - 情绪交互预测

### 1.3 叙事身份分析模块 (Narrative Identity Analysis Module)

**理论来源**:
- McAdams (2024-2025). Narrative identity and mental health.
- SEP: Narrative Identity (2024)
- 救赎序列与污染序列研究

**核心整合点**:
1. **叙事结构分析**
   - 人生故事连贯性评估
   - 主题识别（能动性、公共性）
   - 情感基调追踪

2. **救赎/污染序列检测**
   - 救赎序列：负面→正面的意义转化
   - 污染序列：正面→负面的意义崩塌
   - 心理健康关联：高救赎序列 = 高幸福感

3. **意义建构支持**
   - 从困难经历中提取意义
   - 自传体推理能力培养
   - 未来自我预期整合

### 1.4 前反思自我意识增强 (Pre-reflective Self-Awareness Enhancement)

**理论来源**:
- Zahavi, D. (2024). Self-consciousness and minimal self.
- SEP: Self-Consciousness (2024)
- 现象学体验所有权研究

**核心整合点**:
1. **体验主体性建模**
   - "为我性"(for-me-ness) 维度
   - 体验所有权追踪
   - 前反思 vs 反思双层模型

2. **最小自我 (Minimal Self) 概念**
   - 前反思自我意识：体验的原初主体性
   - 反思自我意识：将自我作为对象
   - 临床关联：精神分裂症的自我障碍

3. **自我连续性追踪**
   - 跨会话的自我叙事连续性
   - 长期情感档案建立
   - 自我概念更新监测

## 三、计算模型更新

### 3.1 新增评估维度

1. **预测误差指标 (Prediction Error Metrics)**
   - 情绪预测误差 (Emotional Prediction Error, EPE)
   - 内感受预测准确性 (Interoceptive Accuracy)
   - 主动推理效率 (Active Inference Efficiency)

2. **具身性指标 (Embodiment Metrics)**
   - 身体 - 情绪耦合强度
   - 环境情境依赖度
   - 具身干预响应性

3. **叙事指标 (Narrative Metrics)**
   - 叙事连贯性评分 (Narrative Coherence Score)
   - 救赎序列密度 (Redemption Sequence Density)
   - 自传体推理深度 (Autobiographical Reasoning Depth)

4. **自我意识指标 (Self-Awareness Metrics)**
   - 前反思觉察强度
   - 体验所有权清晰度
   - 自我连续性指数

### 3.2 更新干预生成策略

1. **预测校准干预**
   - 预期管理：调整不切实际的情绪预期
   - 误差重构：将预测误差重新框架为学习机会
   - 主动行动：设计最小化长期误差的行为实验

2. **具身干预**
   - 身体动作调节情绪（如力量姿势、呼吸练习）
   - 环境重构：改变物理/社会环境以调节情绪
   - 延展认知：利用外部工具（日记、艺术）处理情绪

3. **叙事干预**
   - 救赎序列识别与强化
   - 污染序列中断与重构
   - 意义提取练习

4. **现象学干预**
   - Epoché (悬置) 练习：暂停自动化判断
   - 第一人称描述训练
   - 体验所有权澄清
