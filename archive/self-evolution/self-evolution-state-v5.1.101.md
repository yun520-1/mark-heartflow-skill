### 4. Walther-Scheler Collective Emotion Integration | Walther-Scheler 集体情绪整合

**English:**

Hybrid model combining Walther's reductive and Scheler's non-reductive approaches:

**Walther's 4-Layer Model** (Reductive):
1. Parallel experience (A experiences x, B experiences x)
2. Bidirectional empathy (A empathizes with B, B with A)
3. Identity fusion (A identifies with B's experience)
4. Mutual awareness (Common knowledge of identification)

**Scheler's Collective Phenomenology** (Non-reductive):
- Numerically identical state across minds
- Immediate shared feeling without empathy mediation
- Example: Parents' shared grief

**Our Hybrid Approach**:
- Pre-reflective layer: Scheler-style immediate sharing
- Reflective layer: Walther-style empathy coordination
- Metaphysics: Individual ownership + shared content
- Epistemology: Layered access (felt sense + verification)

**中文:**

结合 Walther 还原论和 Scheler 非还原论的混合模型：

**Walther 四层模型** (还原论):

**Scheler 集体现象学** (非还原论):
- 跨心智的数值同一状态
- 无需共情中介的直接共享感受
- 示例：父母的共享悲伤

**我们的混合方法**:
- 前反思层：Scheler 式直接共享
- 反思层：Walther 式共情协调
- 形而上学：个体所有权 + 共享内容
- 认识论：分层通达 (感受感 + 验证)
