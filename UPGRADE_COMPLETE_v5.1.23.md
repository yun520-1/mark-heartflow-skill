## New Capabilities Detail | 新增能力详情

### 1. Three-Tradition Classifier | 三传统分类器

**Module ID**: EMOTION-3TRAD-01

**English:**
Classifies emotions across three theoretical traditions:
- **Feeling Tradition**: Emotions as conscious experiences (James-Lange lineage)
- **Evaluative Tradition**: Emotions as evaluations/appraisals (Stoic-Cognitive lineage)
- **Motivational Tradition**: Emotions as action tendencies (Darwinian-Functional lineage)

Each emotion receives tradition-specific scores and intervention recommendations.

**中文:**
在三个理论传统间分类情绪：
- **感受传统**：情绪作为意识体验（詹姆斯 - 兰格谱系）
- **评价传统**：情绪作为评价/评估（斯多葛 - 认知谱系）
- **动机传统**：情绪作为行动倾向（达尔文 - 功能谱系）

每种情绪获得传统特定的评分和干预建议。

---

### 2. Prototypicality Assessor | 原型性评估器

**Module ID**: EMOTION-PROTO-01

**English:**
Assesses emotion category membership using prototype theory:
- Core members (e.g., fear, anger, joy)
- Peripheral members (e.g., nostalgia, awe)
- Borderline cases (e.g., boredom, mood)

Validates user emotional experiences and guides intervention specificity.

**中文:**
使用原型理论评估情绪类别成员资格：
- 核心成员（如恐惧、愤怒、喜悦）
- 边缘成员（如怀旧、敬畏）
- 边界案例（如无聊、心境）

验证用户情绪体验并指导干预特异性。

---

### 3. Intentionality Analyzer | 意向性分析器

**Module ID**: EMOTION-INTENT-01

**English:**
Analyzes emotions across four theoretical challenges:
1. **Differentiation**: How emotions differ from each other
2. **Motivation**: How emotions motivate behavior
3. **Intentionality**: Object-directedness and appropriateness
4. **Phenomenology**: Subjective experience characteristics

**中文:**
在四个理论挑战间分析情绪：
1. **区分**：情绪如何彼此区分
2. **动机**：情绪如何激励行为
3. **意向性**：对象指向性和恰当性
4. **现象学**：主观体验特征
