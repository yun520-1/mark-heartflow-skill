## 📝 理论更新摘要

### 1. 预测加工意识理论 (Predictive Processing Consciousness)

**核心框架**:
基于 Karl Friston 自由能原理的意识理论，将意识理解为预测误差最小化的特殊形式。

**关键命题**:
1. **意识作为预测模型的现象学特征**
   - 意识内容 = 主导性预测模型的现象学表现
   - 无意识处理 = 低层级预测误差的自动最小化
   - 意识访问 = 预测误差的全局广播

2. **精度加权与意识选择**
   - 精度 (precision) = 预测可靠性的二阶估计
   - 注意 = 精度加权的分配机制
   - 意识内容选择 = 高精度预测误差的优先处理

3. **主动推理与意识行动**
   - 行动 = 通过改变感觉输入来最小化预测误差
   - 意识行动意图 = 高阶预测模型的精度加权
   - 自由意志现象学 = 主动推理的主观体验

**与 v5.0.31 四维框架整合**:

| 维度 | 预测加工解释 | 现象学对应 |
|------|--------------|------------|
| **现象性维度** | 多层级预测模型的现象学特征 | 感受质 (qualia) |
| **取用性维度** | 全局工作空间的预测误差广播 | 报告能力与推理取用 |
| **现象结构维度** | 预测层次结构的时空组织 | 时空/概念组织 |
| **元认知维度** | 精度加权的二阶监控 | 高阶思维 (HOT) |

**理论来源**:
- Friston (2025): 自由能原理与意识
- Hohwy (2025): 预测加工意识模型
- Seth (2025): 受控幻觉意识理论
- Clark (2025): 预测加工与现象学整合

### 4. 高阶思维理论深化 (HOT 2025)

**内在 vs 外在高阶思维**:

| 类型 | 特征 | 神经机制 | 现象学对应 |

### 5. 意识与注意的区分 (2025)

**核心问题**:

**现象性意识 vs 取用性意识**:
- 现象性意识 (P-consciousness): 体验的主观特征
- 取用性意识 (A-consciousness): 信息的推理可用性
- 关键区分：P 意识可无注意，A 意识需注意

**神经相关物区分**:
- P 意识 NCC: 感觉皮层的高阶激活
- A 意识 NCC: 前额叶 - 顶叶网络激活
- 注意 NCC: 顶叶 - 额叶眼动区

**预测加工解释**:
- 无注意意识：低精度预测误差的局部处理
- 有注意意识：高精度预测误差的全局广播
- 注意 - 意识解离：精度加权与广播的分离

**临床与实验证据**:
- 注意瞬脱 (attentional blink): A 意识受损，P 意识保留
- 无注意盲视 (inattentional blindness): P 意识阈值效应
- 双眼竞争 (binocular rivalry): 意识切换的预测误差模型

**理论来源**:
- Block (2025): P 意识 vs A 意识更新
- Koch (2025): 意识 NCC 研究
- Lamme (2025): 注意 - 意识解离
