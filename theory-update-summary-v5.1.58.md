### 1. Embodied Predictive Processing v2.0 | 具身预测处理 v2.0

**Source | 来源**: Friston (2010, 2012) + Clark (2013, 2016) + Hohwy (2013) + SEP Predictive Processing (2026) + SEP Embodied Cognition (2026)

**English:**

Embodied Predictive Processing v2.0 represents a major theoretical advancement, integrating the Free Energy Principle with the 4E cognition framework (Embodied, Embedded, Enactive, Extended). This module models emotion as hierarchical predictive regulation across multiple levels:

**Theoretical Foundations | 理论基础**:

1. **Free Energy Principle (Friston)**: All self-organizing systems minimize variational free energy—a measure of surprise or prediction error.

2. **Hierarchical Predictive Processing**: The brain implements a hierarchical generative model that predicts sensory inputs at multiple levels of abstraction.

3. **4E Cognition Framework**:
   - **Embodied**: Cognition depends on bodily states and sensorimotor systems
   - **Embedded**: Cognition is situated in environmental context
   - **Enactive**: Cognition emerges through action-perception cycles
   - **Extended**: Cognition extends beyond the brain to tools and others

4. **Active Inference**: Organisms act to fulfill predictions, not just minimize errors—action changes the world to match expectations.

**Key Algorithms | 关键算法**:
```
FREE ENERGY MINIMIZATION | 自由能最小化
┌─────────────────────────────────────────────────────────────┐
│ F = D_KL[Q(s)||P(s|o)] - ln P(o)                           │
│                                                             │
│ Where:                                                      │
│ - F: Free energy (surprise bound)                           │
│ - Q(s): Approximate posterior (beliefs)                     │
│ - P(s|o): True posterior (generative model)                 │
│ - P(o): Prior probability of observations                   │
│                                                             │
│ Minimization via:                                           │
│ 1. Perceptual inference (updating beliefs)                  │
│ 2. Active inference (changing observations via action)      │
└─────────────────────────────────────────────────────────────┘
```

### 4. Precision Weighting & Attention v1.0 | 精度加权与注意 v1.0

**Source | 来源**: Feldman & Friston (2010) + Hohwy (2012) + Clark (2013) + Neuromodulation Research (2015-2026)

**English:**

Precision Weighting theory explains attention as the optimization of confidence in prediction errors versus prior beliefs:

**Theoretical Framework | 理论框架**:

1. **Precision Definition**: Inverse variance (certainty) of prediction errors or priors.

2. **Attentional Gain**: Precision weighting amplifies or suppresses prediction error signals.

3. **Neuromodulatory Mechanisms**:
   - **Dopamine**: Precision of policy selection (action confidence)
   - **Serotonin**: Precision of outcome predictions (reward/punishment)
   - **Norepinephrine**: Precision of sensory prediction errors (alertness)
   - **Acetylcholine**: Precision of sensory input vs. predictions (attention)

4. **Clinical Patterns**:
   - **Anxiety**: Over-weighting of threat-related prediction errors
   - **Depression**: Under-weighting of positive prediction errors
   - **ADHD**: Unstable precision estimation
   - **Psychosis**: Mis-attribution of precision to irrelevant signals

**Precision Weighting Algorithm | 精度加权算法**:
```
PRECISION OPTIMIZATION | 精度优化
┌─────────────────────────────────────────────────────────────┐
│ γ = precision (inverse variance)                            │
│                                                             │
│ Prediction Error Weighting:                                 │
│ weighted_PE = γ_PE × PE                                     │
│                                                             │
│ Prior Weighting:                                            │
│ weighted_prior = γ_prior × prior                            │
│                                                             │
│ Optimal Balance:                                            │
│ γ_PE / γ_prior = reliability_ratio                          │
│                                                             │
│ Neuromodulatory Control:                                    │
│ - Dopamine → γ_policy                                       │
│ - Serotonin → γ_outcome                                     │
│ - Norepinephrine → γ_sensory                                │
│ - Acetylcholine → γ_attention                               │
└─────────────────────────────────────────────────────────────┘
```
