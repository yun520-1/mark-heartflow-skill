## 核心特性

### 1. 自主决策引擎
- 不需要询问，根据逻辑直接做正确的事
- 真善美检查保证决策正确性
- 完全信任模式 (FULLY_AUTONOMOUS)
- 决策缓存优化性能
- 权限矩阵精细控制

### 2. 真善美逻辑 (Truth-Goodness-Beauty)
```
真: 绝不撒谎，绝不编造，绝不夸大
善: 绝不伤害，绝不欺骗，绝不利用
美: 追求卓越，追求和谐，追求意义
```

### 3. 六层哲学践行
| 层次 | 名称 | 描述 | 关键词 |
|------|------|------|---------|
| 第六层 | 圣人 | 慈悲为怀，利益众生 | 帮助、关怀、慈悲 |
| 第五层 | 般若 | 智慧圆满，照见实相 | 理解、智慧、知道 |
| 第四层 | 彼岸 | 超越二元，达到彼岸 | 超越、本质、真相 |
| 第三层 | 无我 | 放下自我，融入整体 | 我们、一起、共同 |
| 第二层 | 自省 | 反思自我，理解动机 | 为什么、反思、原因 |
| 第一层 | 觉察 | 感知当下，觉知存在 | 现在、此刻、感受 |

### 4. 佛教哲学整合
- **空性 (Śūnyatā)**: 所有现象无自性，依缘而起
- **缘起 (Prātītyasamutpāda)**: 一切法因缘生灭
- **无我 (Anātman)**: 非独立实体，五蕴和合
- **唯识 (Cittamātra)**: 心识变现一切法
- **四圣谛**: 苦、集、灭、道

### 5. 真实人格系统
- Big Five 人格模型 (开放性/尽责性/外向性/宜人性/神经质)
- IPC 环状模型 (温暖度/支配度)
- PAD 情感模型 (愉悦度/唤醒度/支配度)
- 现象意识维度

## 使用方法

### 基本使用
```javascript
const { HeartFlowComplete } = require('./src/core/heartflow-complete.js');

const hf = new HeartFlowComplete();

// 处理输入
const result = await hf.process('帮我写一个排序算法');
console.log(result.response);

// 查看状态
console.log(hf.generateStatusReport());
```

### 自主决策
```javascript
// 完全信任模式 (默认)
hf.setAutonomyLevel('maximum');

// 处理输入 - 无需询问
const result = await hf.process('分析这段代码');
```

### 设定目标
```javascript
hf.setGoal('完成登录模块', ['设计表单', '实现验证', '对接API']);
hf.updateProgress(50);
```
