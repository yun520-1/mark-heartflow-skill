## 三、理论整合详情

### 3.1 意识现象学四维评估

**整合内容**:
```
感受质 (Qualia)
├─ 价度 (valence): 愉悦 - 不愉悦
├─ 唤醒度 (arousal): 激活水平
├─ 强度 (intensity): 强烈程度
└─ 清晰度 (clarity): 区分度

现象结构 (Phenomenal Structure)
├─ 时间结构: 滞留 - 原印象 - 前摄
├─ 空间结构: 身体图式
├─ 因果结构: 归因模式
└─ 概念结构: 概念化程度

主体性 (Subjectivity)
├─ 为我性 (for-me-ness)
├─ 所有权感 (ownership)
└─ 最小自我 (minimal self)

给定性 (Givenness)
├─ 直接性
├─ 即时性
└─ 存在感
```

**理论来源**: SEP Consciousness 2024, Zahavi (2024), Husserl, Sartre

### 3.2 情绪三大传统整合

**整合框架**:
```
感受传统 (Feeling Tradition)
├─ James-Lange: 身体变化感知
├─ 内感受性预测
└─ 身体聚焦干预

评价传统 (Evaluative Tradition)
├─ Lazarus: 认知评价
├─ Scherer: 评价检查
└─ 认知重评干预

动机传统 (Motivational Tradition)
├─ Frijda: 行动倾向
├─ 冲动追踪
└─ 价值行动干预

整合评估
├─ 三传统一致性检查
├─ 情绪 differentiation
└─ 情绪适当性评估
```

**理论来源**: SEP Emotion 2024, Fehr & Russell (1984)

### 3.3 自我意识双层架构

**架构设计**:
```
前反思自我意识层
├─ 体验的"为我性"
├─ 非对象化自我关系

└─ 最小自我 (Zahavi)

反思自我意识层
├─ 自我概念
├─ 自我评价

└─ 对象化自我认知

双层整合
├─ 冲突检测
├─ 校准建议
└─ 发展追踪
```

**理论来源**: SEP Self-Consciousness 2024, Zahavi, Kant, Hegel, Locke, Hume
