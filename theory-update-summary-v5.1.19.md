## 1. Embodied Predictive Emotion Processing | 具身预测情绪加工

### 1.1 Theoretical Foundation | 理论基础

**English:**

The Embodied Predictive Emotion Processing module integrates three theoretical frameworks:

1. **Predictive Processing (Friston 2010-2026)**: The brain as a hierarchical prediction machine minimizing prediction error through active inference
2. **Embodied Cognition (Newen, de Bruin & Gallagher 2026)**: Cognition is constituted by dynamic body-environment coupling
3. **Emotion Theory (Scarantino 2016, SEP Emotion)**: Emotions as multi-component processes with feeling, evaluative, and motivational dimensions

**Integration Formula:**
```
Emotion_State = Predictive_Model(Body_State, Environmental_Context, Prior_Emotional_Disposition)
Prediction_Error = Actual_Sensory_Input - Predicted_Sensory_Input
Active_Inference = Minimize(Prediction_Error) through Action or Model_Update
```

**中文:**

具身预测情绪加工模块整合三个理论框架：

1. **预测加工 (Friston 2010-2026)**: 大脑作为层级预测机器，通过主动推理最小化预测误差
2. **具身认知 (Newen, de Bruin & Gallagher 2026)**: 认知由动态身体 - 环境耦合构成
3. **情绪理论 (Scarantino 2016, SEP Emotion)**: 情绪作为具有感受、评价和动机维度的多成分过程

**整合公式:**
```
情绪状态 = 预测模型 (身体状态，环境情境，先前情绪倾向)
预测误差 = 实际感觉输入 - 预测感觉输入
主动推理 = 通过行动或模型更新最小化 (预测误差)
```

## 2. Temporal Self-Consciousness Integration | 时间自我意识整合

### 2.1 Theoretical Foundation | 理论基础

**English:**

Temporal Self-Consciousness integrates:

1. **Husserlian Time-Consciousness (SEP Temporal Consciousness)**: Three-fold structure of retention-primal impression-protention
2. **William James' Specious Present**: The experienced "now" as duration-block with temporal thickness
3. **Predictive Temporal Modeling**: Brain's temporal prediction horizon extending ~3-5 seconds into future
4. **Self-Consciousness Layers (SEP Self-Consciousness)**: Pre-reflective vs. reflective self-awareness

**Integration Model:**
```
Temporal_Self = {
  Retention: {duration: 2-3s, content: fading_awareness},
  Primal_Impression: {duration: ~30ms, content: present_moment},
  Protention: {duration: 3-5s, content: anticipated_future},
  Self_Layer: {pre_reflective: givenness, reflective: thematic_awareness}
}
```

**中文:**

时间自我意识整合：

1. **胡塞尔时间意识 (SEP Temporal Consciousness)**: 保留 - 原印象 - 预期的三重结构
2. **威廉·詹姆斯的显似现在**: 作为持续块的体验"现在"，具有时间厚度
3. **预测时间建模**: 大脑时间预测视界延伸至未来约 3-5 秒
4. **自我意识层级 (SEP Self-Consciousness)**: 前反思与反思自我觉察

**整合模型:**
```
时间自我 = {
  保留：{持续时间：2-3 秒，内容：渐隐觉察},
  原印象：{持续时间：~30 毫秒，内容：当下时刻},
  预期：{持续时间：3-5 秒，内容：预期未来},
  自我层级：{前反思：给定性，反思：主题觉察}
```

**English:**

Collective Emotion Phenomenology synthesizes:

1. **Scheler's Collective Emotion (1954 [1912])**: Emotions can be numerically identical across multiple minds
2. **Walther's Shared Experience (1923)**: Four-layer structure of empathy-identification-mutual awareness
3. **Gilbert's Joint Commitment (2014)**: Collective intentionality as mutual obligation structure
4. **Searle's We-Intention (1990)**: Irreducible collective intentional states

**Integration Framework:**
```

  Scheler_Layer: {type: unity_of_feeling, example: parental_grief},
  Walther_Layer: {empathy: level_3, identification: mutual, awareness: level_4},
  Gilbert_Layer: {commitment: joint, obligation: mutual, enforcement: social},
  Searle_Layer: {intention: we-form, irreducible: true}
}
```

**中文:**

集体情绪现象学综合：

2. **Walther 共享体验 (1923)**: 共情 - 认同 - 相互觉察的四层结构
3. **Gilbert 联合承诺 (2014)**: 集体意向性作为相互义务结构
4. **Searle 我们意向 (1990)**: 不可还原的集体意向状态

**整合框架:**

```
  Scheler 层：{类型：感受统一性，示例：父母哀伤},
```

## 4. Emotion Prototype Structure Refinement | 情绪原型结构精细化

### 4.1 Theoretical Foundation | 理论基础

**English:**

Emotion Prototype Structure builds on:

1. **Fehr & Russell (1984)**: Emotions as prototype categories with graded membership
2. **SEP Emotion §1**: Five-component emotion model (evaluative, physiological, phenomenological, expressive, behavioral)
3. **Barrett's Conceptual Act Theory (2017)**: Emotion construction from conceptual knowledge
4. **Emotional Granularity Research**: Individual differences in emotion differentiation

**Prototype Scoring Model:**
```
Emotion_Prototype = {
  category: 'anger' | 'fear' | 'joy' | ...,
  components: {
    evaluative: {value: 0-1, weight: 0.25},
    physiological: {value: 0-1, weight: 0.20},
    phenomenological: {value: 0-1, weight: 0.25},
    expressive: {value: 0-1, weight: 0.15},
    behavioral: {value: 0-1, weight: 0.15}
  },
  typicality_score: weighted_sum(components),
  confidence: based_on_typicality_and_context,
  borderline_case: typicality_score < threshold
}
```

**中文:**

情绪原型结构基于：

1. **Fehr & Russell (1984)**: 情绪作为具有等级成员关系的原型类别
2. **SEP Emotion §1**: 五成分情绪模型（评价、生理、现象学、表达、行为）
3. **Barrett 概念行为理论 (2017)**: 从概念知识构建情绪
4. **情绪粒度研究**: 情绪区分中的个体差异

**原型评分模型:**
```
情绪原型 = {
  类别：'愤怒' | '恐惧' | '喜悦' | ...,
  成分：{
    评价：{值：0-1, 权重：0.25},
    生理：{值：0-1, 权重：0.20},
    现象学：{值：0-1, 权重：0.25},
    表达：{值：0-1, 权重：0.15},
    行为：{值：0-1, 权重：0.15}
  },
  典型性评分：加权求和 (成分),
  置信度：基于典型性和情境,
  边界案例：典型性评分 < 阈值
}
```
