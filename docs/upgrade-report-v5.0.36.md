## 📝 理论更新摘要

### 1. 无意识知觉理论
**核心发现**:
- 盲视现象证明知觉 without 意识
- 背侧通路 (行动) vs 腹侧通路 (意识)
- 预测误差第 1-3 层可无意识处理
- 全局广播是意识访问阈值

**HeartFlow 整合**:
```
Layer 0-2: 无意识特征提取与模式匹配
Layer 3: 前意识行动准备
Layer 4+: 意识访问 (全局工作空间)
```

### 2. 梦意识现象学
**核心发现**:
- REM 梦：自上而下预测主导
- 预测误差容忍度提高
- 元认知抑制 (除清醒梦)
- 自我模型流动但最小自我保留

**HeartFlow 整合**:
- 梦状态 = 高层预测 without 底部校正
- 清醒梦 = 第 6 层元认知重新激活
- 意识状态谱系连续体

### 3. 阈下加工理论
**核心发现**:
- 阈下启动效应 (语义/情绪/知觉)
- 内隐学习：统计规律提取
- 程序性记忆：基底节 - 小脑回路

**HeartFlow 整合**:
- 阈下启动 = 第 1-3 层预测准备
- 内隐学习 = 先验形成自动化

### 4. 内隐认知架构
**核心发现**:
- 4E 内隐维度扩展
- 内隐 - 外显交互梯度
- 身体图式自动更新

**HeartFlow 整合**:
- 内隐 4E: 具身/嵌入/延展/生成
- 外显系统 (L4-6) ←→ 内隐系统 (L0-3)
