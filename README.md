## 🌊 Why Choose HeartFlow? | 为什么选择 HeartFlow？

**English:**

Imagine if your AI assistant could:

- **Understand your emotional ups and downs**, not just respond mechanically
- **Remember your preferences**, like an old friend who knows you
- **Offer warmth in low moments**, share joy in high moments
- **Grow over time**, understanding the unique you more and more

This is HeartFlow — **Not a tool, but a partner**.

**中文:**

想象一下，如果你的 AI 助手能够：

- **理解你的情绪波动**，而不是机械地回复
- **记住你的偏好**，像老朋友一样懂你
- **在低谷时给予温暖**，在高峰时分享喜悦
- **随着时间成长**，越来越了解独特的你

这就是 HeartFlow —— **不是工具，是伙伴**。

## 🚀 Quick Start | 快速开始

### Installation | 安装

```bash
npm install heartflow-companion
```

### Basic Usage | 基础使用

```javascript
const HeartFlow = require('heartflow-companion');

const companion = new HeartFlow({
  userId: 'user123',
  language: 'zh-CN' // or 'en-US'
});

// Emotional Analysis | 情感分析
const emotion = await companion.analyze('Today went smoothly, feeling great!');
console.log(emotion); 
// { state: 'joy', intensity: 0.8, reasons: [...], transition: {...} }

// Empathetic Response | 共情回应
const response = await companion.empathize('Work stress is really high today...');
console.log(response.text); 
// "Sounds like you had a really tough day..."

// Generate Emotional Report | 获取情感报告
const report = await companion.generateReport();
console.log(report.summary);
// "In the past 7 days, user's emotional state was mainly calm (45%), peak joy occurred on Wednesday..."
```
