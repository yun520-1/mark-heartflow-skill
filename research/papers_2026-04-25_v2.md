## 一、依恋理论 (Attachment Theory)

### 1.1 理论核心框架

依恋理论由**John Bowlby**提出，强调儿童与主要照料者之间形成的深层情感纽带，这种纽带会影响个体后来的社会、情感和认知发展。

**核心概念：内在工作模型（Inner Working Model）**
- 心理表征系统，包含对**自我、他人和关系**的认知图式
- 由早期依恋经历塑造
- 引导个体一生中的人际关系处理方式
- 基于对依恋人物可获得性的信心程度

### 1.2 依恋类型（四分法）

| 类型 | 核心特征 | 关系表现 |
|------|----------|----------|
| **安全型（Secure）** | 信任他人，对亲密关系感到舒适 | 能自如建立亲密关系 |
| **焦虑型（Anxious）** | 不安全感，害怕被抛弃 | 过度依赖、情绪波动 |
| **回避型（Avoidant）** | 情感距离，过度自我依赖 | 回避亲密、恐惧承诺 |
| **混乱型（Disorganized）** | 缺乏明确应对策略 | 矛盾行为、创伤相关 |

### 1.3 2025-2026年最新研究发现

#### 研究1：依恋类型与神经质人格的交互作用
- **来源**: *Child and Adolescent Social Psychiatry and Development*, 2025年4月
- **核心发现**: 约会关系质量受依恋风格与神经质的交互影响。不安全型依恋结合高神经质会显著降低关系满意度。

#### 研究2：依恋与生命史策略的整合
- **来源**: *Evolutionary Psychology*, 2025年7月
- **核心发现**: 交配努力（mating effort）和养育努力（parenting effort）独立中介依恋对浪漫关系的影响。

#### 研究3：依恋测量方法收敛性检验
- **来源**: *Frontiers in Psychology*, 2025年4月
- **核心发现**: 
  - AA I（成人依恋访谈）、BBE（关系特定依恋量表）、IAT（隐含联想测验）三种方法之间相关性**低到中等**
  - 问卷与内隐测量之间相关性 r = 0.21-0.33
  - 同一研究传统内的收敛性高于不同传统之间

#### 研究4：共情压力与成人依恋
- **来源**: *Nature Scientific Reports*, 2025年9月
- **核心发现**: 浪漫伴侣中的共情压力与依恋类型存在显著关联，安全型依恋在共情压力下表现更稳定。

#### 研究5：依恋类型与早期不适应图式
- **来源**: *Personality and Individual Differences*, 2025年12月
- **核心发现**: 安全型依恋显著减少早期不适应图式（EMS）的发生率和强度，焦虑型和回避型则增加风险。

## 二、社会交换理论 (Social Exchange Theory)

### 2.1 理论起源

社会交换理论（Social Exchange Theory, SET）源于**George Homans**的行为主义社会学，结合了经济学成本-收益分析��社会心理学的人际关系研究。

### 2.2 核心概念

| 概念 | 定义 |
|------|------|
| **收益（Rewards）** | 关系中获得的正向体验：情感支持、资源、社会认可 |
| **成本（Costs）** | 关系中付出的代价：时间、精力、情绪消耗、机会成本 |
| **比较水平（Comparison Level, CL）** | 个体基于过去经验对关系的期望收益水平 |
| **比较水平替代选项（CLalt）** | 可替代关系的质量感知 |
| **公平交换（Equity）** | 双方投入比例与收益比例的对等 |

### 2.3 关系动力机制

```
关系满意度 = 实际收益 - 比较水平（CL）
关系稳定性 = 实际收益 - 比较水平替代选项（CLalt）

当 实际收益 > CL → 满意
当 实际收益 > CLalt → 承诺
```

### 2.4 2025-2026年最新研究

- **核心发现**: 知识隐藏行为受社会交换关系质量的调节，低质量交换关系增加知识隐藏倾向。

#### 研究3：领导力研究中的社会交换

### 3.1 理论基石

进化心理学将进化论应用于心理学研究，核心假设：**人类心理机制是自然选择和性选择的结果**。

### 3.2 核心概念

| 概念 | 说明 |
|------|------|
| **配偶价值（Mate Value, MV）** | 综合外表、健康、社会地位、资源的竞争力评估 |
| **配偶价值匹配（MV Matching）** | 双方MV的匹配度影响关系稳定性 |
| **亲代投资（Parental Investment）** | 雌雄在后代养育上的投资差异 |
| **情绪-唤醒理论** | 生理唤醒状态被错误归因于吸引力 |
| **相似效应** | 相似度高增加吸引力 |
