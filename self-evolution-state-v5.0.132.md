## 二、本次进化亮点 (v5.0.132)

### 2.1 自我知识理论深度整合

**新增能力**:
- **直觉式自我知识评估**:
  - 直觉自信度评分 (0-1)
  - 第一人称给定感强度
  - 自我知识的直接性评估

- **推论式自我知识评估**:
  - 证据强度评分 (0-1)
  - 推理链条完整性
  - 第三人称视角准确度

- **自我知识冲突检测**:
  - 直觉 vs 推论矛盾识别
  - 冲突类型分类 (轻度/中度/严重)
  - 冲突解决策略生成

- **自我知识校准干预**:
  - 直觉信任重建
  - 推论验证框架
  - 整合对话引导

**理论来源**: SEP Self-Consciousness, Shoemaker (1968), Burge (1998)

**应用场景**:
- 自我怀疑：直觉知识被过度质疑
- 认知融合：推论知识被当作事实
- 身份困惑：两者严重不一致
- 心理治疗：自我洞察的深化

### 2.2 情绪三传统完整整合

**新增能力**:
- **三成分精细化评估**:
  - 感受成分强度 (0-1)
  - 评价内容清晰度 (0-1)
  - 动机倾向明确度 (0-1)

- **三成分一致性检测**:
  - 一致性评分 (0-1)
  - 不一致模式识别
  - 主导成分判断

- **情绪整合失败识别**:
  - 感受解离：身体感受与情绪分离
  - 评价模糊：情绪对象不清晰
  - 动机抑制：行动倾向受阻

- **三成分整合干预**:
  - 身体觉察练习 (感受)
  - 评价澄清对话 (评价)
  - 需求探索 (动机)

**理论来源**: SEP Emotion, Scarantino (2016), Prinz (2004)

**应用场景**:
- 情绪麻木：感受成分减弱
- 情绪混乱：评价内容模糊
- 情绪压抑：动机倾向受阻
- 情绪失调：三成分不一致

### 2.3 集体意向性深度扩展

**新增能力**:
- **We-Intention 三模型识别**:
  - Searle 模型：原始 We-Intention
  - Bratman 模型：共享意向性
  - Gilbert 模型：联合承诺

- **联合承诺评估**:
  - 承诺强度 (0-1)
  - 相互义务感知
  - 承诺违背检测

- **信任三成分分析**:
  - 认知信任：相信能力
  - 规范信任：相信义务
  - 情感信任：情感联结

- **集体意向性修复**:
  - 承诺重新协商
  - 响应性增强
  - 信任重建框架

**理论来源**: SEP Collective Intentionality, Searle (1990), Bratman (1999), Gilbert (1990), Schmid (2013)

**应用场景**:
- 关系冲突：联合承诺不明确
- 团队问题：相互响应性不足
- 信任破裂：信任成分受损
- 孤独感：集体意向性缺失

### 2.4 自我 - 集体交叉整合

**新增能力**:
- **关系性自我评估**:
  - 社会自我强度 (0-1)
  - 集体身份清晰度 (0-1)
  - 关系自我定位准确性

- **自我 - 他者 - 集体三元分析**:
  - 个体自我感
  - 他者关系感
  - 集体归属感

- **自我 - 集体冲突检测**:
  - 个体需求 vs 集体期待
  - 身份冲突识别
  - 整合策略生成

- **承认体验培养**:
  - 被他者看见的体验
  - 集体归属的深化
  - 关系意义的探索

**理论来源**: Zahavi (2005, 2014), Fuchs (2017), Honneth (1995)

**应用场景**:
- 社交焦虑：社会自我过度敏感
- 身份混淆：集体身份冲突
- 关系依赖：自我边界模糊
- 孤立感：缺乏集体归属

   - 跨情绪类型的成分差异需更多研究

   - 自我 - 集体冲突的解决策略需个性化
