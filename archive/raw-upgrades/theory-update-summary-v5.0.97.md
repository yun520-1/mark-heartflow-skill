## 三、计算模型更新

### 3.1 情绪原型匹配算法 v2.1

**更新公式**:
```javascript
// 情绪原型匹配评分 (新增意识维度加权)
function calculateEmotionPrototypeMatch(emotionState, consciousnessProfile) {
  const baseMatch = calculateBasePrototypeMatch(emotionState);
  const consciousnessWeight = {
    phenomenal: 0.35,    // 现象意识权重
    access: 0.25,        // 取用意识权重
    self: 0.25,          // 自我意识权重
    monitoring: 0.15     // 监控意识权重
  };
  
  const consciousnessAdjustedMatch = 
    baseMatch * (
      consciousnessProfile.phenomenal * consciousnessWeight.phenomenal +
      consciousnessProfile.access * consciousnessWeight.access +
      consciousnessProfile.self * consciousnessWeight.self +
      consciousnessProfile.monitoring * consciousnessWeight.monitoring
    );
  
  return consciousnessAdjustedMatch;
}
```
