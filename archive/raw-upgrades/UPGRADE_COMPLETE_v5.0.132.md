## 三、新增干预策略

### 3.1 自我知识干预 (5 种)

1. **直觉信任练习**
   - 目标：增强对直觉式自我知识的信心
   - 适用：自我怀疑、过度质疑

2. **证据检验对话**
   - 目标：通过证据验证自我信念
   - 适用：认知扭曲、自我否定

3. **整合框架引导**
   - 目标：解决直觉与推论的冲突
   - 适用：身份困惑、内在冲突

4. **第一人称觉察**
   - 目标：培养对给定感的敏感度
   - 适用：解离状态、自我感减弱

5. **自我知识校准**
   - 目标：平衡直觉与推论
   - 适用：自我理解不准确

### 3.2 情绪三成分干预 (6 种)

1. **身体扫描增强**
   - 目标：提升内感受觉察
   - 适用：情绪麻木、感受解离

2. **评价内容澄清**
   - 目标：识别情绪的形式对象
   - 适用：情绪混乱、评价模糊

3. **需求探索对话**
   - 目标：发现情绪驱动的核心需求
   - 适用：动机抑制、行动困难

4. **三成分整合练习**
   - 目标：促进整体情绪体验
   - 适用：情绪失调、成分不一致

5. **情绪恰当性评估**
   - 目标：判断情绪与情境的匹配
   - 适用：情绪过度/不足

6. **动机主题识别**
   - 目标：发现情绪背后的行动倾向
   - 适用：行动方向不明

### 3.3 集体意向性干预 (6 种)

1. **联合承诺澄清**
   - 目标：明确关系中的相互期待
   - 适用：关系冲突、期待不明

2. **相互响应性练习**
   - 目标：增强关系中的协调
   - 适用：团队问题、配合不佳

3. **信任重建框架**
   - 目标：修复受损的信任关系
   - 适用：信任破裂、背叛创伤

4. **集体意义创造**
   - 目标：共同构建关系叙事
   - 适用：关系疏离、意义缺失

5. **We-Intention 识别**
   - 目标：明确集体意图类型
   - 适用：集体行动困难

6. **承认体验培养**
   - 目标：被他者看见的体验
   - 适用：孤独感、不被理解

### 3.4 关系性自我干预 (4 种)

1. **关系身份探索**
   - 目标：发现关系中的自我面向
   - 适用：身份混淆、角色冲突

2. **集体身份澄清**
   - 目标：明确群体归属的意义
   - 适用：归属感缺失、身份冲突

3. **自我 - 他者边界工作**
   - 目标：建立健康的分离与联结
   - 适用：关系依赖、边界模糊

4. **自我 - 集体整合**
   - 目标：平衡个体与集体需求
   - 适用：自我 - 集体冲突
