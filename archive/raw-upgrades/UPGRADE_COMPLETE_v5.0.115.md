## 新增理论模块

### 1. 第一人称视角与自身性模块

**理论来源**: SEP Self-Consciousness §1-2

**核心概念**:
- 自身性 (mineness): 体验的"属于我"的特性
- 第一人称给定：无需二阶监控的直接给定性
- 第一人称权威：主体对自身体验的特殊访问权限

**形式化成果**:
```
∀e (Experience(e) → ∃m (Mineness(m) ∧ Has(e, m)))
```

**临床应用**:
- 去人格化干预
- 自我疏离重建
- 正念训练深化

### 2. 情绪原型结构精细化模块

**理论来源**: SEP Emotion §1 (Fehr & Russell 1984)

**核心概念**:
- 原型范畴 vs 经典范畴
- 典型性梯度 (最佳示例→边缘示例)
- 家族相似性维度

**临床应用**:
- 情绪识别训练
- 混合情绪分析
- 情绪粒度提升

### 3. 信任 - 依靠连续体模块

**理论来源**: SEP Collective Intentionality §2 (Schmid, Alonso)

**核心概念**:
- 依靠 (reliance): 基于预测的认知成分
- 信任 (trust): 认知 + 规范 + 情感成分
- 连续体模型：纯粹依靠 ↔ 深厚信任

**临床应用**:
- 关系类型澄清
- 信任建设策略
- 信任修复策略
