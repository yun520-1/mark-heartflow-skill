## Core Theoretical Advances | 焦点理论进展

### 1. Temporal Self-Consciousness Module | 时间自我意识模块

**English:**

Integrates phenomenological and psychological accounts of temporal self-experience:

**Husserl's Triadic Time Structure (SEP Consciousness §4.2)**:
- **Retention**: Immediate past held in present consciousness (not memory, but primary remembrance)
- **Primal Impression**: The living present, the point of actual experience
- **Protention**: Anticipation of immediate future (not prediction, but primary expectation)

**William James' Specious Present**:
- The experienced "now" has duration (~3 seconds)
- Contains both retentional and protentional aspects
- Foundation for temporal continuity of self

**Contemporary Research Integration**:
- Time consciousness assessment algorithms
- Temporal depth measurement
- Time-emotion cross-analysis
- Temporal self-continuity evaluation

**Computational Implementation**:
```
TemporalSelfModel = {
  retention: Window(past_seconds=3, decay=exponential),
  primal_impression: PresentMoment(experience_buffer),
  protention: Anticipation(predictive_processing_output),
  temporal_depth: Metric(integration_span),
  self_continuity: Score(retention-protention_coherence)
}
```

**中文:**

整合自我时间体验的现象学和心理学解释：

**胡塞尔时间三元组（SEP 意识§4.2）**：
- **滞留**：当下意识中保持的直接过去（非记忆，而是原初回忆）
- **原印象**：活的当下，实际体验的点
- **前摄**：对直接未来的预期（非预测，而是原初期待）

**威廉·詹姆斯显似现在**：
- 体验的"现在"有持续时间（约 3 秒）
- 包含滞留和前摄两个方面
- 自我时间连续性的基础

**当代研究整合**：
- 时间意识评估算法
- 时间深度测量
- 时间 - 情绪交叉分析
- 时间自我连续性评估

**计算实现**：
```
时间自我模型 = {
  滞留：窗口 (过去_秒=3, 衰减=指数),
  原印象：当下时刻 (体验_缓冲区),
  前摄：预期 (预测_加工_输出),
  时间深度：指标 (整合_跨度),
  自我连续性：评分 (滞留 - 前摄_连贯性)
}
```

### 2. Intersubjective Emotion Framework | 主体间情绪框架

**English:**

Enhances collective emotion phenomenology with dynamic systems modeling:

**Scheler's Collective Emotion Phenomenology (SEP Collective Intentionality §2.2)**:
- **Fellow-feeling (Mitgefühl)**: Feeling with another, not just feeling for
- **Emotional Infection**: Pre-reflective emotional synchrony
- **Collective Mood**: Shared affective atmosphere
- **Genuine Collective Emotion**: Numerically identical emotional state across multiple subjects

**Walther's Four-Layer Shared Experience**:
1. Individual experience of x by A and B
2. Empathy with other's experience
3. Identification with other's experience
4. Mutual empathetic awareness of identification

**Dynamic Systems Enhancement**:
- Emotional synchrony measurement (physiological, behavioral, expressive)
- Coupling strength assessment between individuals
- Phase transition detection in collective emotional states
- Emergent collective emotion identification

**Computational Implementation**:
```
IntersubjectiveEmotion = {
  fellow_feeling: Score(empathy_depth × emotional_resonance),
  emotional_infection: Metric(pre_reflective_synchrony),
  collective_mood: Atmosphere(shared_affective_tone),
  genuine_collective: State(numerical_identity_across_subjects),
  coupling_strength: Measure(interpersonal_coupling),
  phase_transition: Detector(collective_state_shift)
}
```

**中文:**

用动态系统建模增强集体情绪现象学：

**谢勒的集体情绪现象学（SEP 集体意向性§2.2）**：
- **共感（Mitgefühl）**：与他人一起感受，不仅是为他人感受
- **情绪感染**：前反思情绪同步
- **集体心境**：共享情感氛围
- **真正集体情绪**：多个主体间数值相同的情绪状态

**瓦尔特共享体验四层**：
1. A 和 B 对 x 的个体体验
2. 对他人体验的共情
3. 对他人体验的认同
4. 对认同的相互共情意识

**动态系统增强**：
- 情绪同步测量（生理、行为、表达）
- 个体间耦合强度评估
- 集体情绪状态相变检测
- 涌现集体情绪识别

**计算实现**：
```
主体间情绪 = {
  共感：评分 (共情_深度 × 情绪_共振),
  情绪感染：指标 (前反思_同步),
  集体心境：氛围 (共享_情感_基调),
  真正集体：状态 (跨主体_数值_同一),
  耦合强度：测量 (人际_耦合),
  相变：检测器 (集体_状态_转换)
}
```

- High-context vs. low-context emotional communication
- Dialectical vs. non-dialectical emotional experience
- Emotional complexity preferences (mixed emotions tolerance)

**Integration Framework**:
- Cultural time-emotion mapping
- Cross-cultural emotional intelligence assessment
- Culturally adaptive intervention generation
- Temporal-emotional-cultural cross-analysis
