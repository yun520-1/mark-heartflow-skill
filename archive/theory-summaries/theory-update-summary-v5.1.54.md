### 1. Hard Problem Framework v1.0 | 困难问题框架 v1.0

**English:**
Integration of David Chalmers' Hard Problem of Consciousness formulation:
- **Easy Problems**: Explaining functions, behaviors, cognitive abilities
- **Hard Problem**: Explaining why and how physical processes give rise to subjective experience
- **Explanatory Gap**: The seeming impossibility of deducing phenomenal facts from physical facts
- **Conceivability Argument**: Zombies are conceivable → zombies are possible → physicalism is false
- **Meta-Problem**: Explaining why we think there's a hard problem

### 6. Zombie Argument Analysis v1.0 | 僵尸论证分析 v1.0

**English:**
Integration of philosophical zombie argument framework:
- **Zombie Definition**: Physical duplicate lacking consciousness
- **Conceivability → Possibility**: Core inference challenged
- **Physicalist Responses**: Deny conceivability or deny inference
- **Anti-Zombie Argument**: Anti-zombies also conceivable
- **Epistemic Contact Problem**: How zombies could refer to qualia

**中文:**
整合哲学僵尸论证框架:
- **僵尸定义**: 缺乏意识的物理复制体
- **可设想性 → 可能性**: 核心推论受挑战
- **物理主义回应**: 否认可设想性或否认可推论
- **反僵尸论证**: 反僵尸也可设想
- **认识接触问题**: 僵尸如何指称感受质

**Zombie Argument Structure | 僵尸论证结构:**
```
Conceivability Argument:
1. Zombies are conceivable (僵尸可设想)
2. Conceivable → Possible (可设想 → 可能)
3. Therefore: Zombies are possible (因此：僵尸可能)
4. If zombies possible → Physicalism false (若僵尸可能 → 物理主义假)

Physicalist Counter-Strategies:
- Deny (1): Zombies not genuinely conceivable
- Deny (2): Conceivability ≠ metaphysical possibility
- Deny (4): Zombie possibility compatible with physicalism
```
