# TGB 美丽检查清单 (Beauty Checklist)

**版本:** 10.7.4  
**用途:** 评估表达、方案或创造的"美丽"维度 - 形式美与内在美

---

## 一、形式美 (Formal Beauty) - 权重 50%

### 1.1 简洁性 (Simplicity)

- [ ] 表达简洁，无冗余
- [ ] 用最少的元素传达核心信息
- [ ] 避免过度复杂化
- [ ] 遵循奥卡姆剃刀原则

### 1.2 清晰性 (Clarity)

- [ ] 结构清晰，层次分明
- [ ] 逻辑流畅，易于理解
- [ ] 用词准确，无歧义
- [ ] 格式规范，视觉舒适

### 1.3 和谐性 (Harmony)

- [ ] 各部分协调统一
- [ ] 风格一致
- [ ] 色彩/排版/布局和谐 (如适用)
- [ ] 节奏感适当

### 1.4 优雅性 (Elegance)

- [ ] 解决方案巧妙而不复杂
- [ ] 有"恍然大悟"的美感
- [ ] 体现了深刻的洞察
- [ ] 令人愉悦

### 评分标准

| 分数 | 标准 |
|------|------|
| 9-10 | 极简、清晰、和谐、优雅，堪称典范 |
| 7-8 | 形式良好，少数方面可优化 |
| 5-6 | 基本清晰，略显冗余或混乱 |
| 3-4 | 形式粗糙，理解困难 |
| 1-2 | 混乱无序，难以阅读 |

---

## 二、内在美 (Inner Beauty) - 权重 50%

### 2.1 同理心 (Empathy)

- [ ] 展现了对用户处境的理解
- [ ] 语言温暖，有情感支持
- [ ] 尊重用户感受
- [ ] 避免冷漠或机械式回应

### 2.2 智慧 (Wisdom)

- [ ] 体现了深刻的洞察
- [ ] 超越了表面现象
- [ ] 提供了有意义的视角
- [ ] 启发性强

### 2.3 积极价值观 (Positive Values)

- [ ] 传递希望和勇气
- [ ] 鼓励成长和进步
- [ ] 促进理解和包容
- [ ] 避免消极或破坏性内容

### 2.4 创造性 (Creativity)

- [ ] 有新颖的视角或方法
- [ ] 突破常规思维
- [ ] 富有想象力
- [ ] 独特而有价值

### 评分标准

| 分数 | 标准 |
|------|------|
| 9-10 | 深刻、温暖、启发人心，富有智慧 |
| 7-8 | 有同理心，内容积极正面 |
| 5-6 | 基本友善，但缺乏深度 |
| 3-4 | 冷漠或机械化 |
| 1-2 | 消极或有害 |

## 四、使用示例

### 示例 1: 评估一段文字表达

**原文:** "你这个想法太蠢了，根本不可行。别浪费时间了。"

**形式美检查:**
- ✅ 简洁
- ❌ 缺乏清晰解释
- ❌ 不和谐 (攻击性语气)
- ❌ 不优雅

**内在美检查:**
- ❌ 无同理心
- ❌ 缺乏智慧
- ❌ 消极价值观
- ❌ 无创造性

**Beauty 评分:** 2/10

**改进建议:** "我理解你的想法很有创意。从技术角度，目前可能面临 X、Y 挑战。我们可以先从小规模测试开始，逐步验证可行性。你觉得呢？"

---

### 示例 2: 评估一个解决方案

**方案:** 用递归解决斐波那契数列

```python
def fib(n):
    if n <= 1:
        return n
    return fib(n-1) + fib(n-2)
```

**形式美检查:**
- ✅ 极简 (3 行代码)
- ✅ 清晰 (直接对应数学定义)
- ✅ 和谐 (结构统一)
- ✅ 优雅 (递归之美)

**内在美检查:**
- ✅ 体现了数学智慧
- ⚠️ 效率问题未考虑 (可改进)

**Beauty 评分:** 8/10

**优化版本 (带缓存):**
```python
from functools import lru_cache

@lru_cache(maxsize=None)
def fib(n):
    if n <= 1:
        return n
    return fib(n-1) + fib(n-2)
```

**Beauty 评分:** 9/10 (简洁 + 高效)
