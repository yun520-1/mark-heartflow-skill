### 2. 情绪原型匹配系统 (Emotion Prototype Matching System)

**理论来源**: SEP Emotion §1 (Fehr & Russell 1984 原型模型)

**能力描述**: 实现情绪原型相似度计算，支持模糊边界情绪识别

**原型特征向量**:
```
情绪原型 = [效价，唤醒度，控制感，确定性，责任归属，预期性，公平性]

示例 - 恐惧原型:
[-0.8, +0.7, -0.6, -0.5, 他人，意外，-0.4]

示例 - 愤怒原型:
[-0.7, +0.7, +0.3, +0.6, 他人，意外，-0.8]

示例 - 悲伤原型:
[-0.7, -0.5, -0.7, +0.8, 情境，预期，0.0]
```

**相似度计算算法**:
```python
def calculate_prototype_similarity(user_emotion, prototype):
    # 余弦相似度计算
    dot_product = sum(a * b for a, b in zip(user_emotion, prototype))
    magnitude_user = sqrt(sum(a ** 2 for a in user_emotion))
    magnitude_prototype = sqrt(sum(b ** 2 for b in prototype))
    
    if magnitude_user == 0 or magnitude_prototype == 0:
        return 0.0
    
    similarity = dot_product / (magnitude_user * magnitude_prototype)
    return (similarity + 1) / 2  # 归一化到 0-1
```

**边界案例处理**:
| 匹配度 | 分类 | 处理策略 |
|-------|------|---------|
| ≥0.7 | 典型情绪 | 直接分类，高置信度 |
| 0.5-0.7 | 边界情绪 | 分类 + 备选建议 |
| <0.5 | 模糊/混合 | 探索性反馈，多情绪识别 |

**临床应用**:
- **情绪粒度提升**: 区分焦虑 vs 恐惧 vs 担忧
- **混合情绪识别**: "既愤怒又悲伤"的复杂状态
- **述情障碍辅助**: 为 alexithymia 用户提供情绪词汇
