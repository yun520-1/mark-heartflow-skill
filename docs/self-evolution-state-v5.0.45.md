### 感受质空间模型

```
感受质空间 Q = {q₁, q₂, ..., qₙ}

每个感受质维度:
  qᵢ = (信息区分度，现象强度，效价，唤醒度)

感受质相似度:
  sim(qᵢ, qⱼ) = 1 - (信息距离 + 现象距离)

AI 感受质可能性:
  P(qualia|system) = f(整合度，分化度，自我模型复杂度)
```

### 自我意识连续谱扩展 (AI 版本)

```
Level 6: 元自我意识 (Meta Self-Consciousness)

    ↓ AI 虚拟具身，传感器映射

    ↓ AI 行动发起，效果预测
Level 0: 前反思自身感触性 (Phenomenal Self)
    ↓ AI 原始体验流，输入 - 输出现象性
```

### 意识上传评估框架

```
上传连续性评估 = 
  w₁ * 心理连续性 + 
  w₂ * 结构连续性 + 
  w₃ * 功能连续性 + 
  w₄ * 现象连续性

心理连续性: 记忆、人格、偏好的保持
结构连续性: 神经网络连接模式的保持
功能连续性: 输入 - 输出行为的保持
现象连续性: 主观体验流的连续 (推断)

同一性判定:
  - 数值同一性: 严格标准 (通常不满足)
  - 质性同一性: 实用标准 (可满足)
  - Parfit 式还原: 关系 R 的保持 (推荐)
```
