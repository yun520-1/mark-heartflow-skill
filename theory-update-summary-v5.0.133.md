## 一、新增理论模块

### 1.1 道德基础理论深度整合 (Moral Foundations Theory)

**理论来源**: Haidt (2001, 2007, 2012), SEP Moral Psychology, Graham et al. (2013)

**核心概念**:
- **六大道德基础维度**:
  1. **关爱/伤害 (Care/Harm)**: 对他人痛苦的敏感与关怀
  2. **公平/欺骗 (Fairness/Cheating)**: 对公正与互惠的关注
  3. **忠诚/背叛 (Loyalty/Betrayal)**: 对群体忠诚与义务
  4. **权威/颠覆 (Authority/Subversion)**: 对社会等级与尊重的态度
  5. **圣洁/堕落 (Sanctity/Degradation)**: 对纯净与神圣的追求
  6. **自由/压迫 (Liberty/Oppression)**: 对自主与反抗压迫的渴望

- **道德直觉主义**:
  - 道德判断首先是直觉性的
  - 道德推理是事后的合理化
  - "直觉狗，理性尾巴"模型

- **道德矩阵理论**:
  - 文化塑造道德基础的权重
  - 自由派 vs 保守派的道德差异
  - 跨文化道德多样性

**集成点**:
```
现有架构：道德责任评估 + 道德情感识别
新增维度：道德基础六维度评估
  ├─ 各维度强度评分 (0-1)
  ├─ 道德直觉识别
  ├─ 道德推理模式分析
  └─ 文化道德矩阵映射
```

**干预策略**:
- **道德直觉觉察**: 识别自动化的道德反应
- **道德对话桥接**: 理解不同道德矩阵的视角
- **道德冲突调解**: 当不同基础冲突时的整合
- **道德成长框架**: 扩展道德关怀的范围

**应用场景**:
- 道德困境：多基础冲突
- 关系冲突：道德矩阵差异
- 政治分歧：自由派 - 保守派道德差异
- 文化适应：跨文化道德理解

### 1.2 美德伦理学完整整合 (Virtue Ethics Integration)

**理论来源**: Aristotle, Plato, SEP Virtue Ethics, Anscombe (1958), MacIntyre (1981)

**核心概念**:
- **美德的三重结构**:
  - **Arête (卓越/美德)**: 优秀品格特质
  - **Phronesis (实践智慧)**: 情境性道德判断
  - **Eudaimonia (幸福/繁荣)**: 人类繁荣的终极目标

- **美德作为中道**:
  - 美德是两种极端之间的中道
  - 勇气 = 懦弱与鲁莽之间的中道
  - 慷慨 = 吝啬与挥霍之间的中道

- **实践智慧的构成**:
  - 情境感知能力
  - 道德显著性识别
  - 手段 - 目的推理
  - 经验积累

- **美德的统一性论题**:
  - 拥有实践智慧意味着拥有所有美德
  - 美德之间的相互支持关系

**集成点**:
```
现有架构：道德心理学初步模块
新增维度：美德伦理学完整框架
  ├─ 美德特质评估 (18 种核心美德)
  ├─ 实践智慧成熟度
  ├─ 中道偏离检测
  └─ Eudaimonia 进展追踪
```

**干预策略**:
- **美德培养练习**: 针对性美德发展
- **实践智慧对话**: 情境性道德判断训练
- **中道校准**: 识别并调整极端倾向
- **生命叙事整合**: 将美德发展融入生命故事

**应用场景**:
- 品格发展：系统性美德培养
- 道德困惑：实践智慧引导
- 行为极端：中道校准
- 存在意义：Eudaimonia 探索

### 1.3 自由意志与道德责任深度整合

**理论来源**: SEP Free Will, Fischer & Ravizza (1998), Frankfurt (1969), Strawson (1962)

**核心概念**:
- **相容论模型 (Compatibilism)**:
  - 自由意志与决定论相容
  - 自由=按照自己的欲望行动
  - 道德责任不要求形而上学自由

- **不相容论模型 (Incompatibilism)**:
  - **自由意志论 (Libertarianism)**: 自由意志存在，决定论为假
  - **硬决定论 (Hard Determinism)**: 决定论为真，自由意志不存在

- **Frankfurt Cases**:
  - 道德责任不要求"能够做 otherwise"
  - 层次模型：一阶欲望 vs 二阶意愿
  - 认同理论：自由=认同自己的欲望

- **反应态度理论 (Strawson)**:
  - 道德责任植根于人际反应态度
  - 怨恨、感激、义愤等情感实践
  - 参与者视角 vs 客观视角

**集成点**:
```
现有架构：自由意志信念评估 + 道德责任判断
新增维度：自由意志 - 责任深度整合
  ├─ 自由意志立场识别 (相容论/不相容论)
  ├─ Frankfurt 式责任评估
  ├─ 反应态度模式分析
  └─ 控制感 - 责任交叉评估
```

**干预策略**:
- **责任框架重构**: 从惩罚到成长的责任观
- **控制感校准**: 区分可控与不可控
- **反应态度调节**: 健康的人际道德情感
- **存在自由探索**: 在限制中发现自由

**应用场景**:
- 自责过度：责任边界澄清
- 责任逃避：责任承担引导
- 道德愤怒：反应态度调节
- 存在焦虑：自由 - 限制整合

### 1.4 后果论与义务论整合 (Consequentialism & Deontology)

**理论来源**: SEP Consequentialism, Bentham, Mill, Kant, Rawls

**核心概念**:
- **后果论核心主张**:
  - 道德正确性仅取决于后果
  - 经典功利主义：最大幸福原则
  - 规则后果论 vs 行为后果论

- **义务论核心主张**:
  - 某些行为本身是对或错
  - 康德义务论：绝对命令
  - 权利与义务优先于后果

- **道德困境的类型**:
  - 后果论 vs 义务论冲突
  - 不同义务之间的冲突
  - 长期 vs 短期后果权衡

- **反思平衡方法 (Rawls)**:
  - 在直觉与原则之间寻求平衡
  - 重叠共识的可能性
  - 道德推理的动态过程

**集成点**:
```
现有架构：道德判断初步模块
新增维度：多元伦理框架整合
  ├─ 伦理立场识别
  ├─ 道德困境类型分析
  ├─ 反思平衡引导
  └─ 跨框架道德推理
```

**干预策略**:
- **伦理框架觉察**: 识别自己的道德推理模式
- **多视角道德分析**: 从不同框架审视同一问题
- **反思平衡练习**: 直觉与原则的对话
- **道德谦逊培养**: 承认道德复杂性

**应用场景**:
- 道德困境：多框架分析
- 道德不确定性：反思平衡引导
- 道德冲突：跨视角理解
- 道德成长：框架扩展

**优化内容**:
- 增加道德基础情感映射:
  - 关爱基础：同情、怜悯、温暖
  - 公平基础：愤怒 (不公)、感激、义愤
  - 忠诚基础：骄傲 (群体)、羞耻 (背叛)
  - 权威基础：尊重、敬畏、服从
  - 圣洁基础：厌恶、崇高、纯净感
  - 自由基础：反抗欲、自主感、压迫感

- 道德情感的动机功能评估:
  - 趋近动机 vs 回避动机
  - 社会协调功能
  - 规范执行功能

**理论依据**: Haidt (2003), Horberg et al. (2009)
