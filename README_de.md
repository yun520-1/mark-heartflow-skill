## 📦 Installation

### Methode 1: Von GitHub installieren (Empfohlen)
```bash
# Repository klonen
git clone https://github.com/yun520-1/mark-heartflow-skill.git
cd mark-heartflow-skill

# Abhängigkeiten installieren
pip install -r requirements.txt  # falls erforderlich

# In Hermes installieren
cp -r . /Users/apple/.hermes/skills/heartflow/
```

### Methode 2: Vom Fähigkeitsmarkt installieren
```bash
# Hermes-Fähigkeitsverwaltung verwenden
hermes skill install heartflow
```

## 🚀 Schnellstart

### Grundlegende Verwendung
```python
from heartflow import HeartFlow

# Initialisierung
hf = HeartFlow(version="10.10.0")

# Logikprüfung
result = hf.check_logic("All humans are mortal. Socrates is human. Therefore, Socrates is mortal.")
print(result)  # ✅ Logik korrekt

# Selbstkorrektur
corrected = hf.self_correct("Socrates is mortal because he is human.")
print(corrected)  # Korrigierte Logik
```

### Integration in Hermes
In `config.yaml` hinzufügen:
```yaml
skills:
  - name: heartflow
    path: /Users/apple/.hermes/skills/heartflow/
    enabled: true
```
