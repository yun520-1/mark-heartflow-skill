### Flow Theory Integration | 心流理论整合

**English:**

Flow theory is integrated with phenomenological self-consciousness and predictive processing:

**Flow Characteristics**:
1. Complete absorption (pre-reflective awareness)
2. Challenge-skill balance (optimal prediction error)
3. Clear goals (intentional structure)
4. Immediate feedback (prediction error signaling)
5. Effortlessness (skilled embodied coping)
6. Temporal distortion (altered time consciousness)
7. Autotelic experience (intrinsic motivation)

**Flow Channels**:
- Flow: High challenge + High skill
- Anxiety: High challenge + Low skill → Intervention: Skill building
- Boredom: Low challenge + High skill → Intervention: Increase challenge
- Apathy: Low challenge + Low skill → Intervention: Engage in meaningful activity

**Integration with Existing Modules**:
- Flow ↔ Pre-Reflective Self (Zahavi): Flow as non-objectified self-awareness
- Flow ↔ Embodied Cognition: Flow as skillful bodily engagement
- Flow ↔ Predictive Processing: Flow as optimal prediction error minimization
- Flow ↔ Temporal Consciousness: Time distortion as altered retention-protention structure

**中文:**

心流理论与现象学自我意识和预测加工整合：

**心流特征**：
1. 完全沉浸（前反思觉察）
2. 挑战 - 技能平衡（最优预测误差）
3. 清晰目标（意向结构）
4. 即时反馈（预测误差信号）
5. 毫不费力（熟练具身应对）
6. 时间扭曲（改变的时间意识）
7. 自目的体验（内在动机）

**心流通道**：
- 心流：高挑战 + 高技能
- 焦虑：高挑战 + 低技能 → 干预：技能建立
- 无聊：低挑战 + 高技能 → 干预：增加挑战
- 冷漠：低挑战 + 低技能 → 干预：参与有意义的活动

**与现有模块整合**：
- 心流 ↔ 前反思自我（Zahavi）：心流作为非对象化自我意识
- 心流 ↔ 具身认知：心流作为熟练身体参与
- 心流 ↔ 预测加工：心流作为最优预测误差最小化
- 心流 ↔ 时间意识：时间扭曲作为改变的原型 - 预期结构

