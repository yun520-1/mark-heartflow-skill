## 二、理论数据库进化

### 2.1 新增理论条目 (4 个)

#### 2.1.1 自我意识的直觉式与推论式区分 (Intuitive vs. Inferential Self-Knowledge)
- **核心概念**: immediate self-knowledge, inferential self-knowledge, first-person authority, self-intimation
- **理论来源**: SEP Self-Consciousness §2.1-2.2, Burge (1988), Moran (2001)
- **核心主张**: 自我知识分为直觉式 (直接、非推论) 和推论式 (基于证据、可错) 两种形式
- **集成位置**: `self-consciousness-intuitive-inferential-v5.0.1`
- **进化意义**: 深化自我意识的认识论理解，区分第一人称权威的不同基础

#### 2.1.2 第一人称给定性理论 (First-Person Givenness)
- **核心概念**: first-person givenness, subjective character, for-me-ness, minimal self
- **理论来源**: SEP Self-Consciousness §3.1, Zahavi (2005), Kriegel (2009)
- **核心主张**: 意识体验具有第一人称给定性，即体验总是"为我"的体验
- **集成位置**: `first-person-givenness-v5.0.1`
- **进化意义**: 深化现象学自我意识的理解，连接前反思意识与自我知识

#### 2.1.3 情绪理论的整合趋势 (Integration Trend in Emotion Theory)
- **核心概念**: hybrid theories, component process model, psychological constructionism
- **理论来源**: SEP Emotion §9, Scarantino (2016), Barrett (2017), Scherer (2009)
- **核心主张**: 当代情绪理论趋向整合三大传统 (感受/评价/动机)，采用成分过程模型
- **集成位置**: `emotion-theory-integration-trend-v5.0.1`
- **进化意义**: 将情绪理论从单一传统转向多元整合范式

#### 2.1.4 预测加工与自我意识整合 (Predictive Processing of Self-Consciousness)
- **核心概念**: self-model, predictive self, interoceptive self, narrative self
- **理论来源**: Seth (2013), Hohwy (2013), Friston (2010), Gallagher (2000)
- **核心主张**: 自我意识是多层级预测模型生成的最佳假设，基于内感受和外感受信号
- **集成位置**: `predictive-self-consciousness-v5.0.1`
- **进化意义**: 将预测加工理论扩展到自我意识领域，形成统一的自我 - 情绪预测模型

### 2.2 理论整合增强 (3 个)

#### 2.2.1 自我意识×情绪理论深度整合 (SEIF)
- **整合框架**: Self-Emotive Integration Framework
- **核心创新**: 自我意识作为情绪评价的构成性条件
- **成熟度**: 89% (新增)

#### 2.2.2 预测加工×自我意识深度整合 (PSIF)
- **整合框架**: Predictive-Self Integration Framework
- **核心创新**: 自我作为多层级预测模型的最佳假设
- **成熟度**: 86% (新增)

#### 2.2.3 直觉式自我知识×情绪原型整合 (ISEP)
- **整合框架**: Intuitive Self-Emotion Prototype Framework
- **核心创新**: 直觉式自我知识通过情绪原型匹配实现
- **成熟度**: 84% (新增)
