## 🔐 Audit de Sécurité v10.4.0

### ✅ Problèmes Corrigés
- Limites de longueur d'entrée (max 50 000 caractères)
- Prévention d'injection HTML/script
- Détection de crise à temps constant
- Itération bornée (prévient ReDoS)
- Opérations de moteur thread-safe
- Pas de données sensibles dans les messages d'erreur
- Vérification de limites d'entiers type-safe
- Limites de ressources sur toutes les collections

## 📦 Installation Universelle

```bash
git clone https://github.com/yun520-1/mark-heartflow-skill.git
cd mark-heartflow-skill
./install.sh
```

### Utilisation Basique

```python
from heartflow import HeartFlow, process_input

result = process_input("Aider les autres me rend heureux")
print(result.decision)
print(result.emotion_analysis['primary'])
```
