### 1. Predictive Processing & Active Inference | 预测加工与主动推理

**Capability | 能力**: Implement Friston's Free Energy Principle for emotional prediction, prediction error computation, and precision-weighted attention modulation

**Implementation | 实现**:
```javascript
// Hierarchical predictive processing model
{
  predictiveProcessing: {
    hierarchicalModel: {
      level5_conceptual: {
        prediction: "Social situation will be threatening",
        prior: 0.7,  // strength of belief
        abstraction: "high"
      },
      level4_appraisal: {
        prediction: "Others will judge me negatively",
        prior: 0.65,
        abstraction: "medium-high"
      },
      level3_emotion: {
        prediction: "Anxiety will arise",
        prior: 0.75,
        abstraction: "medium"
      },
      level2_interoceptive: {
        prediction: {
          heartRate: 95,
          respiration: 22,
          muscleTension: 0.7
        },
        prior: 0.8,
        abstraction: "medium-low"
      },
      level1_sensory: {
        prediction: "Tightness in chest, rapid breathing",
        prior: 0.85,
        abstraction: "low"
      }
    },
    
    predictionError: {
      level5: 0.15,  // conceptual error
      level4: 0.22,  // appraisal error
      level3: 0.18,  // emotion error
      level2: 0.25,  // interoceptive error
      level1: 0.30,  // sensory error
      weightedTotal: 0.22  // precision-weighted
    },
    
    precisionWeighting: {
      attention: "threat-focused",
      precisionLevel: 0.8,  // high precision = hypervigilance
      modifiable: true
    },
    
    intervention: {
      strategy: "precision_downregulation",
      technique: "open_monitoring_meditation",
      instruction: "Notice sensations without narrowing attention to threat",
      expectedPrecisionChange: -0.3
    }
  }
}
```

**Key Features | 关键特性**:
- **Hierarchical Prediction**: 5-level generative model from conceptual to sensory
- **Precision Modulation**: Attention as precision weighting on prediction errors
- **Active Inference**: Action selection to minimize expected prediction error
- **Interoceptive Inference**: Emotional feelings as predictions about bodily states

### 3. Moral Emotion & Social Norm Internalization | 道德情绪与社会规范内化

**Capability | 能力**: Recognize and respond to moral emotions with guilt-shame differentiation and norm internalization stage assessment

**Implementation | 实现**:
```javascript
// Moral emotion assessment
{
  moralEmotion: {
    primary: "guilt",  // or "shame", "anger", "disgust", "elevation", etc.
    
    category: "self-conscious",  // other-condemning / self-conscious / other-suffering / other-praising
    
    appraisal: {
      focus: "behavior",  // guilt: behavior; shame: global self
      thought: "I lied to my friend",  // specific action
      vs: "I am a liar",  // shame: global self
      responsibility: 0.90,
      controllability: 0.85,
      normViolation: "honesty"
    },
    
    actionTendency: {
      primary: "reparation",  // guilt: repair; shame: hide
      behaviors: [
        "Apologize sincerely",
        "Make amends",
        "Commit to honesty"
      ],
      vs: ["Withdraw", "Hide from friend", "Avoid confrontation"]  // shame
    },
    
    normInternalization: {
      stage: "identified",  // external / introjected / identified / integrated
      motivation: "Personal value of honesty",
      vs: "Avoid guilt" (introjected) or "I'm an honest person" (integrated)
    },
    
    moralFoundations: {
      primary: "fairness",  // care / fairness / loyalty / authority / sanctity / liberty
      violation: "Cheating/deception violates fairness norm"
    },
    
    intervention: {
      forGuilt: [
        "Facilitate sincere apology",
        "Support reparation planning",
        "Encourage behavioral commitment"
      ],
      forShame: [
        "Separate behavior from identity",
        "Practice self-compassion",
        "Challenge 'I am bad' belief",
        "Reconnect with supportive others"
      ]
    }
  }
}
```

**Key Features | 关键特性**:
- **Moral Emotion Taxonomy**: Four categories with 12+ specific emotions
- **Guilt-Shame Differentiation**: Behavioral vs. global self focus assessment
- **Norm Internalization Stages**: External → introjected → identified → integrated
- **Moral Foundations**: Haidt's six foundations for value assessment

**能力**：识别和响应道德情绪，包含内疚 - 羞耻差异化和规范内化阶段评估

**关键特性**：
- **道德情绪分类**：四个类别，12+ 具体情绪
- **内疚 - 羞耻差异化**：行为 vs. 整体自我焦点评估
