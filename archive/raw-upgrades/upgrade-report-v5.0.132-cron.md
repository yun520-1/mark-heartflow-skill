   - 核心理论：We-Intention 三模型、联合承诺、信任

### 2.2 理论提取

#### 自我意识理论关键点

```
1. 直觉式自我知识 (Immediate Self-Knowledge)
   - 无需推理或观察即可获得的自我知识
   - "我思"的直接给定性
   - 第一人称权威的基础

2. 推论式自我知识 (Inferential Self-Knowledge)
   - 通过观察、推理获得的自我知识
   - 第三人称视角的自我理解
   - 心理治疗中的自我洞察

3. 自我知识的冲突
   - 直觉式 vs 推论式的矛盾
   - 整合策略：双层验证模型

4. 第一人称给定性 (First-Person Givenness)
   - 体验的"为我性" (for-me-ness)
   - 现象学给定感的不可还原性
```

#### 情绪理论关键点

```
1. 感受传统 (Feeling Tradition)
   - James-Lange 理论的现代诠释
   - 身体反馈的情绪构成作用
   - 内感受觉察 (Interoceptive Awareness)

2. 评价传统 (Evaluative Tradition)
   - 情绪的评价内容理论
   - 形式对象 (Formal Object) 概念
   - 情绪的恰当性评估

3. 动机传统 (Motivational Tradition)
   - 情绪的行动倾向理论
   - 核心动机主题识别
   - 情绪驱动的行为模式

4. 三传统整合模型
   - 情绪=感受 + 评价 + 动机的动态整合
   - 各成分的相对权重因情绪类型而异
   - 整合失败=情绪障碍
```

#### 集体意向性关键点

```
1. We-Intention 三模型
   - Searle: 原始 We-Intention
   - Bratman: 共享意向性
   - Gilbert: 联合承诺

2. 集体意向性的构成条件
   - 相互响应性 (Mutual Responsiveness)
   - 承诺性 (Commitment)
   - 共同知识 (Common Knowledge)
   - 相互义务 (Mutual Obligation)

3. 信任作为基础 (Schmid)
   - 认知成分：相信他人会履行
   - 规范成分：有义务履行
   - 情感成分：情感性信任
```
