## 📝 理论更新摘要

### 1. 情绪元认知双层模型

**核心区分**: 前反思情绪自我意识 vs 反思情绪自我意识

| 维度 | 前反思情绪自我意识 | 反思情绪自我意识 |
|------|-------------------|-----------------|
| **意识类型** | 情绪体验的直接给予性 | 对情绪状态的对象化认知 |
| **现象特征** | "我正在生气"的即时感 | "我为什么生气"的分析 |
| **时间结构** | 当下情绪流 | 情绪回溯与预期 |
| **认知资源** | 自动、低消耗 | 控制、高消耗 |
| **神经基础** | 岛叶 + 杏仁核 + 默认网络 | 前额叶 + 前扣带回 |

**形式化表达**:
```
Pre-reflective Emotional SC = ∀E ∈ Emotional_Experience: has_affective_mineness(E)
Reflective Emotional SC = ∃t: attend(emotional_state_as_object, t)
```

### 2. 情感预测误差理论

**核心公式**:
```
APE = | Predicted_affective_state - Actual_affective_state |

APE 层次:
- Level 1: 内感受预测误差 (身体状态)
- Level 2: 情感范畴预测误差 (情绪分类)
- Level 3: 社会情感预测误差 (社会评价)
- Level 4: 自我情感预测误差 (情感 - 自我一致)
```

**现象学映射**:
| APE 水平 | 现象学体验 | 行为倾向 |
|---------|-----------|---------|
| APE < θ1 (低) | 情绪流畅、情感透明 | 情绪表达自然 |
| θ1 ≤ APE < θ2 (中) | 轻微情绪觉察 | 情绪调节启动 |
| θ2 ≤ APE < θ3 (高) | 明显情绪困扰 | 强烈调节努力 |
| APE ≥ θ3 (极高) | 情绪混乱、情感解体 | 调节失败、情绪爆发 |

### 3. 情绪 - 自我 - 元认知三元统一框架

**统一架构**:
```
Level 5: 元元认知
    ↓
Level 4: 元认知 (含情绪元认知)
    ↓
Level 3: 反思情绪自我意识 ←→ 对象认知层 ←→ 前反思情绪自我意识
    ↓
Level 2: 情感预测层 (内感受 - 情感回路)
    ↓
Level 1: 内感受输入层
    ↓
Level 0: 感官输入
```

**关键整合机制**:
- 情感精度调制认知精度
- 情绪触发反思切换
- 自我模型调制情感预测

### 4. 情绪调节的预测加工重释

**Gross 过程模型的预测加工解释**:

| 策略 | 预测加工解释 |
|------|-------------|
| 情境选择 | 选择情感预测误差预期的情境 |
| 情境修改 | 修改情境预测模型 |
| 注意分配 | 调整预测误差计算的感觉权重 |

| 反应调制 | 调整情感预测误差的行为输出 |

**调节效率公式**:
```
调节效率 = (APE_reduction - Regulation_cost) / Baseline_APE
```

### 核心算法伪代码

```javascript
// 情感预测误差计算
function calculateAffectivePredictionError(currentState, affectiveModel) {
  const predictedAffect = affectiveModel.predict(currentState.context);
  const actualAffect = decodeInteroceptiveSignals(currentState.bodySignals);
  
  const apeInteroceptive = divergence(predictedAffect.body, actualAffect.body);
  const apeCategorical = divergence(predictedAffect.category, actualAffect.category);
  const apeSocial = divergence(predictedAffect.social, actualAffect.social);
  const apeSelf = divergence(predictedAffect.self, actualAffect.self);
  
  const apeTotal = weightedSum([apeInteroceptive, apeCategorical, apeSocial, apeSelf], 
                                currentState.contextWeights);
  
  // 根据 APE 大小决定情绪意识类型
  if (apeTotal > REFLECTIVE_THRESHOLD) {
    return { type: 'reflective-emotional', intensity: apeTotal };
  } else {
    return { type: 'pre-reflective-emotional', intensity: apeTotal };
  }
}

// 情绪监测
function emotionMonitoring(emotionalState, context) {
  return {
    emotionLabel: classifyEmotion(emotionalState.valence, emotionalState.arousal),
    confidence: calculateConfidence(emotionalState.evidence),
    intensity: emotionalState.intensity,
    uncertainty: calculateEntropy(emotionalState.categoryDistribution)
  };
}

// 情绪控制
function emotionControl(monitoringResult, strategies) {
  const utilities = strategies.map(s => ({
    strategy: s,
    efficiency: (monitoringResult.apeReduction - s.cost) / monitoringResult.baselineAPE
  }));
  
  return argmax(utilities, u => u.efficiency);
}

// 社会情感预测误差
function calculateSocialAffectivePE(selfModel, otherEvaluation, context) {
  const predictedEvaluation = selfModel.predictSocialEvaluation(context);
  const actualEvaluation = otherEvaluation;
  
  const socialAPE = divergence(predictedEvaluation, actualEvaluation);
  
  // 社会情绪分类
  if (socialAPE > 0 && actualEvaluation < predictedEvaluation) {
    return { emotion: 'shame', intensity: socialAPE };
  } else if (socialAPE < 0 && actualEvaluation > predictedEvaluation) {
    return { emotion: 'pride', intensity: -socialAPE };
  } else if (actualEvaluation > selfModel.selfEvaluation) {
    return { emotion: 'envy', intensity: socialAPE };
  } else {
    return { emotion: 'gratitude', intensity: -socialAPE };
  }
}
```
