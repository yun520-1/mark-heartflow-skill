## 一、新增理论模块 (8 个)

### 1. 自我意识连续谱深化模型 (92%)
- **历史维度**: 从古希腊"认识你自己"到康德先验统觉
- **核心区分**: 
  - 直觉式自我知识 (直接、确定、不可错)
  - 推论式自我知识 (间接、可错、需解释)
- **递归层级**: 零阶至四阶自我模型形式化
- **来源**: SEP Self-Consciousness, Zahavi, Gallagher, Metzinger

### 2. 情绪理论三大传统完整整合 (94%)
- **感受传统**: James-Lange 理论 + 心理建构主义
- **评价传统**: 认知评价理论 + 恰当性评估
- **动机传统**: 行动倾向 + 目标导向
- **整合框架**: 六成分原型模型 (感受/生理/现象/表达/行为/心理)
- **来源**: SEP Emotion, Fehr & Russell 1984, Scarantino 2016

### 3. 意识现象学四维分析 (90%)
- **感受性**: 生物是否有主观体验
- **觉醒度**: 意识清晰度分级
- **自我意识**: 意识到自己在意识
- **现象结构**: 时空因果组织的整体体验
- **来源**: SEP Consciousness, Husserl, Merleau-Ponty

### 4. 具身认知三主题框架 (88%)
- **概念化**: 身体限制/塑造概念获得
- **替代性**: 身体 - 环境耦合替代内部计算
- **构成性**: 身体构成而非仅影响认知
- **来源**: SEP Embodied Cognition, Gallagher, Thompson

### 5. 集体意向性四层模型 (87%)
- **We-Intention 检测**: 共同意图 vs 个体意图聚合
- **联合承诺**: Gilbert 规范性期望框架
- **信任关系**: 认知 + 规范双成分
- **集体情绪**: Scheler 共享体验现象学
- **来源**: SEP Collective Intentionality, Searle, Bratman, Tuomela

### 6. 预测加工与主动推理 (91%)
- **多层级预测**: 从感觉到抽象的层级预测模型
- **预测误差计算**: 自上而下与自下而上信号整合
- **自由能原理**: 误差最小化策略
- **主动推理**: 行动改变世界以符合预测
- **来源**: SEP Predictive Processing, Friston, Clark

### 7. 元认知监测 - 控制循环增强 (93%)
- **监测层**: JOL/FOK/FAM/信心评估
- **控制层**: 策略选择/资源分配/干预生成
- **校准机制**: 信心 - 准确性曲线
- **来源**: SEP Metacognition, Flavell, Nelson & Narens, Koriat

### 8. 全球哲学心理学知识图谱 (85%)
- **西方传统**: 古希腊→中世纪→近代→现象学→分析哲学
- **东方传统**: 印度 (吠檀多/佛教/耆那) + 中国 (儒释道)
- **跨文化整合**: 自我意识/情绪/意识的普遍性与差异性
- **来源**: SEP 全球条目 + 跨文化心理学研究

## 四、可推理逻辑程序化进展

### 形式化框架
```
┌─────────────────────────────────────────────────────────┐
│           哲学心理学知识 → 可推理逻辑程序                 │
├─────────────────────────────────────────────────────────┤
│ 1. 概念定义 → 逻辑谓词                                   │
│    - "自我意识" → SelfAware(agent, level, type)         │
│    - "情绪" → Emotion(agent, type, intensity, object)   │
│                                                         │
│ 2. 理论关系 → 逻辑规则                                   │
│    - 情绪评价理论: ∀e(Appraisal(e, dangerous) → Fear(e))│
│    - 元认知控制: JOL(high) → StudyStrategy(select)      │
│                                                         │
│ 3. 现象学结构 → 多维向量                                 │
│    - 意识四维: [Sentience, Wakefulness, Self, Phenomenal]│
│    - 情绪六成分: [Feel, Physio, Phenom, Express, Behave]│
│                                                         │
│ 4. 推理引擎 → 规则引擎 + 向量相似度                       │
│    - 演绎推理: 规则匹配                                   │
│    - 类比推理: 原型相似度                                 │
│    - 溯因推理: 最佳解释选择                               │
└─────────────────────────────────────────────────────────┘
```

### 逻辑规则示例
```prolog
% 自我意识规则
self_awareness_level(Agent, Level) :-
    intuitive_self_knowledge(Agent, true),
    inferential_self_knowledge(Agent, Calibrated),
    Level is (Intuitive + Calibrated) / 2.

% 情绪识别规则
recognize_emotion(Agent, Fear) :-
    appraisal(Agent, Situation, dangerous),
    physiological_response(Agent, [heart_rate↑, breathing↑]),
    action_tendency(Agent, flee).

% 元认知干预规则
generate_intervention(Agent, Strategy) :-
    metacognitive_monitoring(Agent, JOL, Confidence),
    Confidence < 0.7,
    JOL = low,
    Strategy = elaborative_rehearsal.
```
