### 2. 意识测量指标体系

**三级指标架构**:

**一级指标 (核心维度)**:
- LoC (Level of Consciousness): Φ值总量
- CoC (Content of Consciousness): 因果结构模式
- CI (Consciousness Integration): Φ_Eff/Φ_max
- CB (Consciousness Boundary): 排他性阈值

**二级指标 (功能维度)**:
- PWE (Precision Weighting Efficiency): 广播范围×精度
- PES (Prediction Error Sensitivity): 误差检测率
- MCA (Meta-Cognitive Accuracy): 监控准确性
- SMS (Self-Model Stability): 叙事连贯性

**三级指标 (临床维度)**:
- 意识障碍筛查: Φ_Eff < 0.5 预警
- 麻醉深度监测: 实时Φ值追踪
- 睡眠质量评估: 夜间Φ值曲线
- 冥想进展追踪: 长期ΔΦ变化

### 3. 全局工作空间定量分析

**全局可及性指数 (GAI)**:
```
GAI = (激活模块数 / 总模块数) × 平均精度权重
```

| GAI 范围 | 意识状态 | 行为表现 |
|----------|----------|----------|
| 0.8-1.0 | 完全清醒 | 最佳认知功能 |
| 0.6-0.8 | 正常清醒 | 日常功能 |
| 0.4-0.6 | 疲劳/分心 | 注意力下降 |
| 0.2-0.4 | 困倦/冥想 | 意识内容减少 |
| <0.2 | 睡眠/麻醉 | 意识丧失 |

**神经相关物映射**:
- 全局工作空间 → 前额叶 - 顶叶网络 (EEG 伽马同步)
- 专门处理器 → 感觉/运动皮层 (局部振荡功率)
- 广播机制 → 丘脑皮层回路 (功能连接强度)
- 注意系统 → 背侧注意网络 (精度加权信号)

### 4. 预测加工-IIT 统一框架

**理论映射**:
| IIT 4.0 概念 | 预测加工对应 | 数学形式 |
|--------------|--------------|----------|
| 因果力 | 预测影响力 | p(未来状态\|当前模型) |
| 不可还原性 | 全局模型优势 | F_global < ΣF_local |
| 特定性 | 预测误差模式 | argmin PE(pattern) |
| 统一性 | 精度加权整合 | Σ(精度×预测) |
| 边界性 | 误差阈值 | PE > θ → 无意识 |

**自由能 - Φ关系**:
```
F = -ln p(感觉) + KL[后验||先验]
Φ ∝ 1/F (在稳态条件下)
```
推论：最小化自由能 ↔ 最大化整合信息
