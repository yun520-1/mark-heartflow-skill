## 一、升级目标

基于 v5.0.64 的情感计算与自我意识深化基础，v5.0.65 聚焦于**社会认知与心理理论 (Theory of Mind)** 的整合，实现从个体情感到社会交互的扩展。

### 核心目标
1. 整合心理理论 (ToM) 框架，支持对他人心智状态的推理
2. 深化社会认知模型，包括共情、社会学习、群体动力学
3. 扩展自我意识的社会维度 (集体自我、关系自我)
4. 建立社会 - 情感预测循环，支持社会交互中的预测处理

## 五、关键哲学基础整合

### 5.1 他心问题 (Problem of Other Minds)
- **传统类比论证**: 基于自我 - 他人相似性的推理
- **现象学进路**: 直接感知他人心智 (Zahavi 2014)
- **预测处理进路**: 他心作为社会预测模型 (新整合)

### 5.2 社会认知哲学
- **第二人称进路**: 社会交互作为第一性 (Reddy 2008)
- **参与式意义建构**: 社会理解作为共同创造
- **关系本体论**: 自我作为关系性存在

### 5.3 心理理论框架
- **理论论 (Theory-Theory)**: ToM 作为朴素心理学理论
- **模拟论 (Simulation Theory)**: ToM 作为自我模拟
- **整合进路**: 预测处理框架下的理论 - 模拟统一

### 7.2 核心 API 扩展
```javascript
// 心理理论推理
const tomResult = heartflow.social.reasonAboutMind({
  agent: 'other_person',
  mentalState: 'belief',
  content: 'the box contains candy',
  order: 1  // 一阶 ToM
});

// 社会预测
const socialPrediction = heartflow.social.predict({
  situation: 'job_interview',
  otherAgent: 'interviewer',
  timeHorizon: 'next_5_minutes'
});

// 共情计算
const empathy = heartflow.social.empathize({
  target: 'friend',
  emotion: 'sadness',
  components: ['cognitive', 'affective', 'concern']
});

// 关系自我激活
const relationalSelf = heartflow.self.getRelationalSelf({
  relationshipType: 'friendship',
  context: 'social_gathering'
});
```
