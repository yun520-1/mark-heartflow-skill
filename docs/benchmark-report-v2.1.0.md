## v2.0 → v2.1.0 变更

### 新增：选择题答案选择（selectAnswer）
- 自动检测推理类型并匹配对应评分规则
- 7类推理规则：演绎、归纳、溯因、条件、数学、概率、统计
- 12类谬误识别：稻草人/滑坡/虚假二分/诉诸权威/循环论证等
- LLM 兜底：规则引擎打0分时自动调用 LLM 推理

### 新增：空间关系推理
- 从"X is to the right/left of Y"建立空间关系图
- 支持直接位置陈述（"X is the rightmost/leftmost"、"X is the second from the left"）
- 处理单条空间关系+直接位置组合

### 修复的 bug
1. 选项提取正则：最后选项末尾不匹配（漏了最后一个选项）
2. 空间关系方向：leftOf/rightOf 语义反了
3. sorted 遍历方向：用 rightOf 而不是 leftOf
4. leftmost/rightmost 计算：优先使用 fixedPositions
5. leftmost 兜底检测：排除 fixedPositions 中声明为 rightmost 的物品
6. 归纳/谬误关键词补充
