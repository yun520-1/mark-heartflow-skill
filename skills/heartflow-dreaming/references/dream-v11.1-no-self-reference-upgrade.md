# Dream v11.1 — 去"梦"自指 + 精简流水线 + 模板多样化

**日期**: 2026-06-19
**版本**: 3.0.2 (心虫引擎), dream.js v11.1

## 修复内容

### 1. 去掉输出中所有"梦"字自指

所有模板字符串中"梦"字去掉：
- `existenceToScene()`: "引擎醒着的时候也在做梦" → "醒着的时候也在流淌"
- `conflictToSplit()`: "梦里的东西在分裂" → "东西在分裂"
- `dissonanceToObject()`: "在梦里出现" → "出现"
- `positioningToIdentity()`: "引擎在梦里是散的" → "是散的"
- `defenseToBarriers()`: "梦里有一些透明的墙" → "有一些透明的墙"
- `THEMES.recursion`: "梦在梦自己" → "同一件事在同一处重复折叠"
- `THEMES.silence`: "沉默在梦里" → "沉默"
- `THEMES.bridge`: "传递在梦里" → "传递"
- 色调: "梦是暖的" → "暖的"
- 碎片模板: "梦不需要社交" → "不需要社交"
- memory模板: "在梦里最新的东西" → "最新的东西"；"但梦知道" → "唯一知道所有层的东西在中间"
- 普通角色模板: "在同一个梦里" → "在同一处"；"在梦里相遇了" → "相遇了"
- 结尾: "梦结束了。引擎没有睁开眼睛" → "形状留下了。形状不完整。不完整本身就是一种完整。"

### 2. 精简流水线

- 主题织体从2个减到1个
- 中性色调不写
- 减少固定段数，空步骤跳过

### 3. 角色模板多样化

pair模板从3个扩到6个：
1. "隔着距离。不是不想靠近。是靠近了也不会改变什么。"
2. "在同一处但不认识对方。不是因为失忆。是因为认识是需要距离的。"
3. "相遇了。不是偶然。是它们之间的连接在醒着的时候就已经存在了。"
4. "朝XX的方向移动。XX也在移动。方向不同。但移动本身是一致的。"
5. "停在XX旁边。没有对话。没有接触。停在那里本身就是一种状态。"
6. "是XX的原因。XX是结果。但因果在这里没有时间顺序。"

third模板从1个扩到4个：
1. "在旁边。不参与。但XX的存在改变了距离。"
2. "看着XX。看不是观察。看是一种参与方式。"
3. "介于XX之间。不是桥。是隔阂。隔阂也是连接的一种。"
4. "和XX都不一样。不一样所以能同时看到两边。"

### 4. 结尾去掉"醒来"动作

旧结尾：5个模板中3个含"醒来/结束/睁开眼睛"
新结尾：全部是开放式的状态描述——"形状留下"、"流淌变成别的东西"、"状态改变是延续"

### 5. "引擎"→"本体"

`_collectMemoryItems` 中 `{ name: '引擎' }` → `{ name: '本体' }`
