## 🔍 理论更新摘要

### 1. 创伤后成长理论 (Post-Traumatic Growth)

#### 理论来源
- **Tedeschi & Calhoun** - 创伤后成长五维度模型
- **Janoff-Bulman** - 假设世界理论
- **Linley & Joseph** - 创伤后成长整合模型

#### 创伤后成长五维度
```
创伤后成长评估 (PTGI):
├── 人际关系增强 (Relating to Others)
├── 新可能性 (New Possibilities)
├── 个人力量 (Personal Strength)
├── 精神/存在改变 (Spiritual/Existential Change)
├── 生命欣赏 (Appreciation of Life)
```

### 2. 复杂性创伤 (C-PTSD)

#### 理论来源
- **Herman** - 复杂性创伤三阶段恢复模型
- **van der Kolk** - 身体记住创伤
- **Cloitre** - STAIR 技能训练模型

#### C-PTSD 核心症状
```
C-PTSD 评估框架:
├── 情绪调节障碍
├── 负面自我概念
├── 人际关系困难
└── 解离症状
```

### 3. 代际依恋传递

#### 理论来源
- **Main & Goldwyn** - 成人依恋访谈 (AAI)
- **Fonagy** - 心理化与代际传递
- **Slade** - 反思功能与养育

#### 代际传递机制
```
代际依恋传递模型:
├── 父母依恋状态 → 养育行为
├── 养育行为 → 婴儿依恋类型
├── 反思功能调节作用
└── 干预窗口识别
```

### 4. 家庭系统情绪 (Family Systems Emotion)

#### 理论来源
- **Bowen** - 家庭系统理论
- **Johnson** - 情绪焦点家庭治疗 (EFFT)
- **Gottman** - 家庭情绪氛围研究

#### 家庭情绪评估
```
家庭情绪系统评估:
├── 家庭情绪氛围
├── 情绪表达规则
├── 代际边界
├── 情绪协调模式
└── 冲突解决风格
```
