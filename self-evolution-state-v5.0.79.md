```
   - 益处：亲社会性、幸福感

2. 艺术/音乐敬畏 (Art/Music Awe)
   - 触发：伟大艺术作品、音乐表演
   - 效价：积极
   - 益处：创造力、审美敏感

3. 道德敬畏 (Moral Awe)
   - 触发：英雄行为、无私善举
   - 效价：积极
   - 益处：道德提升、亲社会动机

4. 宗教/超自然敬畏 (Religious Awe)
   - 触发：宗教体验、超自然感知
   - 效价：混合 (积极/消极)
   - 益处：意义感、归属感

5. 知识敬畏 (Knowledge Awe)
   - 触发：科学发现、深刻洞察
   - 效价：积极
   - 益处：好奇心、学习动机

6. 集体敬畏 (Collective Awe)
   - 触发：集体仪式、社会运动
   - 效价：混合
   - 益处：集体认同、团结感

状态：✅ 完整集成
```
