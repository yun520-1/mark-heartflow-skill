# 原生家庭创伤分析 → 职业发展建议框架

## 触发场景
用户口述原生家庭经历后，要求"分析我心理"+"我未来怎么发展"。

## 核心原则

### 1. 先读两份材料
- 用户的简历PDF（提取工作经历、技能栈、跳槽频率）
- cell skill（原生家庭、心理结构、创伤模式）

### 2. 心理分析要做"对比镜像"
用户的人格往往与创伤来源（通常为母亲/父亲）形成镜像：
- **过度付出 vs 过度索取**：用户拿自己当所有人的垫脚石，创伤来源拿所有人当垫脚石
- **不敢为自己活 vs 只为自己活**：用户在等别人给答案，创伤来源在强迫所有人按她的答案活
- **底层相同**：都没学会"为自己活"

### 3. 职业发展建议的结构

#### 诊断用户的职业状态
| 维度 | 观察点 |
|------|--------|
| 行业位置 | QC底层→中层管理→质量负责人/总监 |
| 核心优势 | 体系能力、客户对接（世界500强）、跨行业 |
| 核心短板 | 学历、跳槽频率、是否在核心产业带 |
| 心理模式 | 是否"在等人给答案" |

#### 给出三条路径
格式：每条路径写清楚「条件 → 障碍 → 时间线 → 收入预期」

**方向一：向上（质量总监/VP）**
- 条件：需要本科学历
- 障碍：学历门槛会被HR筛掉
- 时间线：2.5年（成教/自考）
- 收入：年薪35-50万

**方向二：向外（第三方审核/咨询）**
- 条件：ISO审核员资质
- 障碍：需要先考证
- 时间线：半年考证，一年内转型
- 收入：日薪1500-3000元

**方向三：甲方（品牌方品控经理）**
- 条件：乙方工厂管理经验 + 大客户对接经验
- 障碍：简历需包装突出世界500强客户
- 时间线：半年内
- 收入：年薪30-45万

#### 给出"最重要的只有一件事"
在三条路径之上，给出那个**最核心的单点改变**（通常是一个教育投资或一个资质认证），其余所有路径都依赖这个基础。

### 4. 心理层面的结论要落在"你没问题"
用户问"我未来怎么发展"时，他的潜台词是"我够不够好"。所以分析最后要给出明确的判断：
- 能力没问题
- 问题不在能力，在"谁在定义你的价值"
- 你不是你母亲，你已经走出来了
