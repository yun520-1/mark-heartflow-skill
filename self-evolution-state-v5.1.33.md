## Known Limitations | 已知限制

**English:**

1. **Infinite Recursion Problem**: Walther's four-layer model raises question of whether full common knowledge requires infinite mutual awareness. Current implementation uses 4-layer practical cutoff.

2. **Cross-Cultural Variation**: Self-construal varies across cultures (independent vs. interdependent). Current model based on phenomenological universals; cultural calibration planned for v5.1.34.

3. **Clinical Validation**: Pre-reflective self-disturbances (depersonalization, schizophrenia) require clinical validation. Preliminary detection implemented but not clinically validated.

4. **Collective Emotion Boundaries**: Distinguishing Scheler-type from Durkheim-type in practice can be ambiguous. Current classifier uses probabilistic matching.

**中文:**

1. **无限递归问题**：Walther 四层模型提出完全共同知识是否需要无限相互觉知的问题。当前实现使用四层实际截断。

2. **跨文化变异**：自我建构在跨文化中存在差异（独立自我 vs. 互依自我）。当前模型基于现象学普遍性；文化校准计划于 v5.1.34。

3. **临床验证**：前反思自我紊乱（去人格化、精神分裂症）需要临床验证。初步检测已实现但未临床验证。

4. **集体情绪边界**：实践中区分 Scheler 型与 Durkheim 型可能模糊。当前分类器使用概率匹配。
