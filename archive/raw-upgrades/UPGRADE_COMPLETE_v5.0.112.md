## 版本变更详情

### 新增功能

#### 1. 现象学还原五步干预
- **理论来源**: SEP Self-Consciousness §1-2
- **实现内容**: 悬置预设 → 直接描述 → 本质变更 → 主体极定位 → 时间结构分析
- **应用场景**: 去人格化干预、情绪澄清、自我知识校准

#### 2. 无限递进终止算法
- **理论来源**: SEP Collective Intentionality §2.2
- **实现内容**: 信任双成分评估 + 递进层数监控 + 终止决策
- **应用场景**: 关系 OCD、依恋焦虑、元关系防御

#### 3. 情绪三传统整合评估
- **理论来源**: SEP Emotion §2 (Scaranto 2016)
- **实现内容**: 感受 - 评价 - 动机三维度评估 + 整合分析
- **应用场景**: 情绪识别困难、情绪调节、情绪教育

#### 4. 集体情绪融合度检测
- **理论来源**: SEP Collective Intentionality §2.2 (Scheler, Walther)
- **实现内容**: 情绪同步性 + 认同融合 + 去个体化评估
- **应用场景**: 群体动力学、亲密关系、社会运动分析

#### 5. 自我 - 集体交叉评估矩阵
- **理论来源**: SEP Self-Consciousness × SEP Collective Intentionality
- **实现内容**: 四层交叉评估 + 风险识别 + 干预路径
- **应用场景**: 关系评估、依恋模式识别、社交困难
