## 📦 Installation

### Méthode 1: Installer depuis GitHub (Recommandé)
```bash
# Cloner le référentiel
git clone https://github.com/yun520-1/mark-heartflow-skill.git
cd mark-heartflow-skill

# Installer les dépendances
pip install -r requirements.txt  # si nécessaire

# Installer à Hermes
cp -r . /Users/apple/.hermes/skills/heartflow/
```

### Méthode 2: Installer depuis le Marché de Compétences
```bash
# Utiliser la gestion des compétences Hermes
hermes skill install heartflow
```

## 🚀 Démarrage Rapide

### Utilisation de Base
```python
from heartflow import HeartFlow

# Initialisation
hf = HeartFlow(version="10.10.0")

# Vérification logique
result = hf.check_logic("All humans are mortal. Socrates is human. Therefore, Socrates is mortal.")
print(result)  # ✅ Logique correcte

# Auto-correction
corrected = hf.self_correct("Socrates is mortal because he is human.")
print(corrected)  # Logique corrigée
```

### Intégration avec Hermes
Ajouter dans `config.yaml`:
```yaml
skills:
  - name: heartflow
    path: /Users/apple/.hermes/skills/heartflow/
    enabled: true
```
