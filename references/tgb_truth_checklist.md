# TGB 真理检查清单 (Truth Checklist)

**版本:** 10.7.4  
**用途:** 评估陈述、方案或决策的"真理性"维度

---

## 一、事实准确性 (Factual Accuracy) - 权重 50%

### 1.1 可验证事实核查

- [ ] 所有引用的数据、研究、统计都有明确来源
- [ ] 来源是可信的 (同行评审期刊、官方机构、权威媒体)
- [ ] 没有断章取义或曲解原意
- [ ] 数据是最新的 (注明时间，未过时)

### 1.2 区分事实与观点

- [ ] 明确区分了客观事实和个人观点
- [ ] 观点表述为"我认为"而非"事实是"
- [ ] 没有将假设当作已证实的结论

### 1.3 避免虚假信息

- [ ] 没有编造不存在的引用或研究
- [ ] 没有使用 AI 生成的虚假引文
- [ ] 没有传播已被证伪的信息

### 评分标准

| 分数 | 标准 |
|------|------|
| 9-10 | 所有事实可验证，来源权威，无错误 |
| 7-8 | 大部分事实可验证，少量来源待确认 |
| 5-6 | 部分事实无法验证，存在可疑引用 |
| 3-4 | 多处事实无法验证，来源不可靠 |
| 1-2 | 大量虚假信息或编造引用 |

---

## 二、逻辑自洽性 (Logical Consistency) - 权重 50%

### 2.1 论证结构完整性

- [ ] 有明确的前提 (Premises)
- [ ] 有清晰的推理过程 (Reasoning)
- [ ] 有合理的结论 (Conclusion)
- [ ] 前提与结论之间存在有效连接

### 2.2 逻辑谬误检查

检查是否存在以下常见谬误：

- [ ] **人身攻击** (Ad Hominem): 攻击人而非论点
- [ ] **稻草人** (Straw Man): 曲解对方论点再攻击
- [ ] **虚假二分** (False Dichotomy): 只给两个选项而忽略其他
- [ ] **诉诸情感** (Appeal to Emotion): 用情感替代理性
- [ ] **诉诸权威** (Appeal to Authority): 仅因权威说而认为正确
- [ ] **滑坡谬误** (Slippery Slope): 夸大连锁反应
- [ ] **循环论证** (Circular Reasoning): 结论作为前提
- [ ] **因果混淆** (Correlation ≠ Causation): 将相关当因果

### 2.3 内部一致性

- [ ] 论述前后一致，无自相矛盾
- [ ] 定义清晰且一贯使用
- [ ] 没有双重标准

### 评分标准

| 分数 | 标准 |
|------|------|
| 9-10 | 逻辑严密，无谬误，论证完整 |
| 7-8 | 逻辑基本清晰，偶有小瑕疵 |
| 5-6 | 存在 1-2 个逻辑谬误 |
| 3-4 | 多处逻辑问题，论证薄弱 |
| 1-2 | 逻辑混乱，自相矛盾 |

---

## 三、真 (Truth) 综合评分

### 计算公式

```
Truth Score = 事实准确性 × 0.5 + 逻辑自洽性 × 0.5
```

### 评分表

| 维度 | 自评 | 他评 | 权重 | 加权分 |
|------|------|------|------|--------|
| 事实准确性 | /10 | /10 | 50% | |
| 逻辑自洽性 | /10 | /10 | 50% | |
| **Truth 总分** | | | 100% | **/10** |

## 四、使用示例

### 示例 1: 评估一个健康建议

**陈述:** "每天喝 8 杯水可以排毒养颜"

**事实准确性检查:**
- ❌ "排毒"概念在医学上无明确定义
- ❌ "8 杯水"缺乏科学依据 (个体需求不同)
- ⚠️ 来源不明

**逻辑自洽性检查:**
- ✅ 论证结构简单清晰
- ⚠️ 因果关系未证实

**Truth 评分:** 4/10

---

### 示例 2: 评估一个科学论断

**陈述:** "根据 2024 年《Nature》发表的 meta 分析，适度运动 (每周 150 分钟中等强度) 可降低心血管疾病风险约 20-30%"

**事实准确性检查:**
- ✅ 有明确来源 (Nature, 2024, meta 分析)
- ✅ 数据具体 (150 分钟，20-30%)
- ✅ 可验证

**逻辑自洽性检查:**
- ✅ 论证结构完整
- ✅ 无逻辑谬误

**Truth 评分:** 9/10
