**版本**: v1.0.0

**目标**: 为 HeartFlow 添加数学/物理公式能力，建立公式知识库

---

## 📊 引擎架构

### 1. **公式知识库** (`formulas.json`)
- 存储所有公式的结构化数据
- 格式：
  ```json
  {
    "id": "newton_second_law",
    "name": "牛顿第二定律",
    "category": "classical_mechanics",
    "formula": "F = m * a",
    "latex": "F = m \\cdot a",
    "variables": {
      "F": {"name": "力", "unit": "N", "description": "物体受到的合力"},
      "m": {"name": "质量", "unit": "kg", "description": "物体的惯性质量"},
      "a": {"name": "加速度", "unit": "m/s²", "description": "物体的加速度"}
    },
    "description": "物体的加速度与作用力成正比，与质量成反比",
    "applications": ["运动学", "动力学", "工程力学"],
    "difficulty": "beginner",
    "references": ["Newton, I. (1687). Philosophiæ Naturalis Principia Mathematica"]
  }
  ```

### 2. **公式搜索引擎** (`formula-search.js`)
- 根据关键词/领域/难度搜索公式
- 支持中文/英文搜索
- 返回公式的完整信息（公式、变量、应用、推导）

### 3. **公式计算器** (`formula-calculator.js`)
- 代入数值计算公式结果
- 支持单位转换
- 验证计算结果合理性

### 4. **公式推导器** (`formula-deriver.js`)（未来升级）
- 从基础公式推导复杂公式
- 符号计算（使用 `mathjs` 或 `sympy`)
- 生成推导步骤

### 5. **arXiv 论文下载器** (`arxiv-downloader.js`)（未来升级）
- 自动下载 arXiv 论文 PDF
- 提取论文中的公式（OCR / LaTeX 提取）
- 自动添加到公式知识库

## 📚 公式分类（计划收录）

### 数学
- **代数**: 一元二次方程、因式分解、对数/指数
- **几何**: 勾股定理、三角函数、解析几何
- **微积分**: 导数、积分、微分方程
- **线性代数**: 矩阵运算、特征值、奇异值分解
- **概率统计**: 贝叶斯定理、正态分布、假设检验

### 物理
- **经典力学**: 牛顿三定律、动能定理、动量守恒
- **电磁学**: 库仑定律、安培定律、法拉第电磁感应
- **热力学**: 热力学三定律、理想气体状态方程
- **量子力学**: 薛定谔方程、海森堡不确定性原理
- **相对论**: 洛伦兹变换、质能方程

### 化学
- **化学反应**: 平衡化学方程式、反应速率
- **热力学**: 焓变、熵变、吉布斯自由能
- **量子化学**: 薛定谔方程（分子）、分子轨道理论

### 工程
- **电路**: 欧姆定律、基尔霍夫定律、戴维南定理
- **流体力学**: 伯努利方程、纳维-斯托克斯方程
- **材料力学**: 应力-应变关系、梁的弯曲
