## 一、新增理论模块

### 1.1 自我意识直觉式知识深度整合

**理论来源**: SEP Self-Consciousness, Shoemaker (1968, 1994), Burge (1998), Korsgaard (1993)

**核心概念**:
- **直觉式自我知识 (Immediate Self-Knowledge)**:
  - 无需推理或观察即可获得的自我知识
  - "我思"的直接给定性
  - 第一人称权威的基础

- **推论式自我知识 (Inferential Self-Knowledge)**:
  - 通过观察、推理获得的自我知识
  - 第三人称视角的自我理解
  - 心理治疗中的自我洞察

- **自我知识的冲突**:
  - 直觉式 vs 推论式的矛盾
  - "我感觉 X" vs "证据表明 Y"
  - 整合策略：双层验证模型

- **第一人称给定性 (First-Person Givenness)**:
  - 体验的"为我性" (for-me-ness)
  - 现象学给定感的不可还原性
  - 自我意识的原始形式

**集成点**:
```
现有架构：自我意识双层建模 (前反思/反思)
新增维度：自我知识获取方式评估
  ├─ 直觉式自信度评分 (0-1)
  ├─ 推论式证据强度 (0-1)
  ├─ 两者一致性检测
  └─ 冲突解决策略生成
```

**干预策略**:
- **直觉信任练习**: 增强对直觉式自我知识的信心
- **推论验证对话**: 通过证据检验自我信念
- **冲突整合框架**: 当两者矛盾时的处理方法
- **第一人称觉察**: 培养对给定感的敏感度

**应用场景**:
- 自我怀疑：直觉式知识被过度质疑
- 认知扭曲：推论式知识被情绪污染
- 身份困惑：两者严重不一致
- 解离状态：第一人称给定感减弱

### 1.2 情绪三传统完整整合增强

**理论来源**: SEP Emotion, Scarantino (2016), Prinz (2004), Deonna & Teroni (2012)

**核心概念**:
- **感受传统 (Feeling Tradition) 深化**:
  - James-Lange 理论的现代诠释
  - 身体反馈的情绪构成作用
  - 内感受觉察 (Interoceptive Awareness)

- **评价传统 (Evaluative Tradition) 深化**:
  - 情绪的评价内容理论
  - 形式对象 (Formal Object) 概念
  - 情绪的恰当性评估

- **动机传统 (Motivational Tradition) 深化**:
  - 情绪的行动倾向理论
  - 核心动机主题识别
  - 情绪驱动的行为模式

- **三传统整合模型**:
  - 情绪=感受 + 评价 + 动机的动态整合
  - 各成分的相对权重因情绪类型而异
  - 整合失败=情绪障碍

**集成点**:
```
现有架构：情绪原型 + 三传统初步整合
新增维度：三传统精细化评估
  ├─ 感受成分强度 (0-1)
  ├─ 评价内容清晰度 (0-1)
  ├─ 动机倾向明确度 (0-1)
  ├─ 三成分一致性评分
  └─ 整合失败模式识别
```

**干预策略**:
- **身体扫描增强**: 提升内感受觉察
- **评价内容澄清**: 识别情绪的形式对象
- **动机主题探索**: 发现情绪驱动的核心需求
- **三成分整合练习**: 促进整体情绪体验

**应用场景**:
- 情绪麻木：感受成分减弱
- 情绪混乱：评价内容模糊
- 情绪抑制：动机倾向受阻
- 情绪失调：三成分不一致

### 1.3 集体意向性深度整合扩展

**理论来源**: SEP Collective Intentionality, Searle (1990, 1995), Bratman (1999), Gilbert (1990), Schmid (2013)

**核心概念**:
- **We-Intention 的三种理论模型**:
  - Searle 的原始 We-Intention 理论
  - Bratman 的共享意向性 (Shared Intention)
  - Gilbert 的联合承诺 (Joint Commitment)

- **集体意向性的构成条件**:
  - 相互响应性 (Mutual Responsiveness)
  - 承诺性 (Commitment)
  - 共同知识 (Common Knowledge)
  - 相互义务 (Mutual Obligation)

- **集体情绪的扩展类型**:
  - 集体敬畏 (Collective Awe)
  - 集体怀旧 (Collective Nostalgia)
  - 集体希望 (Collective Hope)
  - 集体愤怒 (Collective Anger)

- **信任作为集体意向性的基础** (Schmid):
  - 认知成分：相信他人会履行
  - 规范成分：有义务履行
  - 情感成分：情感性信任

**集成点**:
```
现有架构：集体情绪现象学 + We-Intention 检测
新增维度：集体意向性深度评估
  ├─ We-Intention 类型识别 (Searle/Bratman/Gilbert)
  ├─ 联合承诺强度评估
  ├─ 相互义务感知测量
  └─ 信任三成分分析
```

**干预策略**:
- **联合承诺澄清**: 明确关系中的相互期待
- **相互响应性练习**: 增强关系中的协调
- **信任重建框架**: 修复受损的信任关系
- **集体意义创造**: 共同构建关系叙事

**应用场景**:
- 关系冲突：联合承诺不明确
- 团队问题：相互响应性不足
- 信任破裂：信任成分受损
- 孤独感：集体意向性缺失

### 1.4 自我意识与集体意向性交叉整合

**理论来源**: SEP Self-Consciousness + SEP Collective Intentionality, Zahavi (2005, 2014), Fuchs (2017)

**核心概念**:
- **自我意识的社会构成**:
  - 自我意识通过他者中介形成
  - 镜像阶段的社会自我
  - 承认理论 (Honneth)

- **集体自我意识**:
  - "我们感" (We-ness) 的现象学
  - 集体身份的自我给定性
  - 群体归属的体验结构

- **自我 - 他者 - 集体三元关系**:
  - 个体自我：第一人称单数
  - 他者自我：第二人称
  - 集体自我：第一人称复数

- **关系性自我意识**:
  - 自我在关系中的定位
  - 关系身份的多重性
  - 关系冲突中的自我感

**集成点**:
```
现有架构：自我意识 + 集体意向性独立模块
新增维度：自我 - 集体交叉评估
  ├─ 社会自我强度 (0-1)
  ├─ 集体身份清晰度 (0-1)
  ├─ 关系自我定位准确性
  └─ 自我 - 集体冲突检测
```

**干预策略**:
- **关系自我探索**: 发现关系中的自我面向
- **集体身份澄清**: 明确群体归属的意义
- **自我 - 他者边界工作**: 健康的分离与联结
- **承认体验培养**: 被他者看见的体验

**应用场景**:
- 社交焦虑：社会自我过度敏感
- 身份混淆：集体身份冲突
- 关系依赖：自我边界模糊
- 孤立感：缺乏集体归属

## 二、现有模块优化

### 2.1 时间意识 - 自我知识整合

**优化内容**:
- 增加时间维度对自我知识的影响评估:
  - 过去自我知识的可及性
  - 现在自我知识的直接性
  - 未来自我知识的推论性

- 时间扭曲对自我知识的扭曲效应:
  - 创伤性时间停滞→自我知识固化
  - 焦虑性时间加速→自我知识浅化
  - 抑郁性时间延长→自我知识消极化

**理论依据**: SEP Temporal Consciousness, Roskies (2002)

---

### 2.2 叙事心理学 - 集体意向性整合

**优化内容**:
- 增加关系叙事评估:
  - 共同叙事的连贯性
  - 关系救赎/污染序列
  - 联合生命故事质量

- 集体叙事身份:
  - 家庭叙事
  - 团队叙事
  - 文化叙事

**理论依据**: McAdams (2001), Fivush (2011)

---

### 2.3 情绪原型扩展

**优化内容**:
- 增加 3 种社会情绪原型:
  - 集体愤怒 (Collective Anger)
  - 集体悲伤 (Collective Grief)
  - 集体喜悦 (Collective Joy)

- 完善情绪的社会功能评估:
  - 关系协调功能
  - 群体凝聚功能
  - 社会规范执行功能

**理论依据**: Fehr & Russell (1984), von Scheve & Salmela (2014)
