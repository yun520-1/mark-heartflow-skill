# Academic Research via OpenAlex

## 用途

搜索哲学/心理学学术论文，支撑"古为今用"升级方法论。
OpenAlex 完全免费无需 key，是心经/哲学/心理学升级的首选学术搜索工具。

## API

**Base**: `https://api.openalex.org`
**格式**: `GET /works?search={query}&filter=publication_year:>{year}&per_page={n}&select=title,authorships,publication_year,cited_by_count,doi,abstract_inverted_index`
