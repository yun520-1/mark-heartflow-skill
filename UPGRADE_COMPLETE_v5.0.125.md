## 理论突破

### 1. 自我意识现象学深化

**核心洞察**：前反思自我意识是体验的内在特征，而非附加的高阶状态。

**理论来源**：
- SEP Self-Consciousness Phenomenological
- Sartre 前反思自我意识理论
- Husserl 时间意识与自我
- 直觉式 vs 推论式自我知识区分

**计算实现**：
```javascript
PrereflectiveAwarenessTracker {
  calculateForMeNess()      // 计算为我性强度
  detectMode()              // 检测前反思/反思模式
  assessFirstPersonGivenness() // 评估第一人称给定性
}
```

**干预应用**：
- 前反思觉察训练 (5-10 分钟)
- 第一人称给定性恢复 (10-15 分钟)

### 2. 情绪三大传统整合

**核心洞察**：情绪是感受、评价、动机三成分的动态整合，符合原型结构。

**理论来源**：
- SEP Emotion (三大传统完整整合)
- James-Lange 感受传统
- Nussbaum/Solomon 评价传统
- Ekman/Darwin 动机传统
- Fehr & Russell 情绪原型理论

**计算实现**：
```javascript
EmotionTriadicIntegrator {
  assessTriad()             // 三成分评估
  calculateConsistency()    // 成分一致性计算
  matchEmotionPrototype()   // 情绪原型匹配
}
```

**干预应用**：
- 情绪三成分一致性检查 (15-20 分钟)
- 情绪原型映射 (10-15 分钟)

### 3. 集体意向性现象学

**核心洞察**：集体情绪不是个体情绪的聚合，而是"同一情绪状态"被多个主体共同经历。

**理论来源**：
- SEP Collective Intentionality
- Scheler 集体情绪现象学 (同一情绪状态)
- Walther 共享体验四层模型
- Searle/Gilbert/Bratman 集体意向性理论

**计算实现**：
```javascript
CollectiveIntentionalityDetector {
  detectSchelerModel()      // Scheler 同一情绪检测
  detectWaltherModel()      // Walther 四层评估
  assessWeIntention()       // We-Intention 强度评估
}
```

**干预应用**：
- 共享体验深化 (20-30 分钟，双人)
- We-Intention 培养 (10-15 分钟)

### 4. 具身认知 4E 框架

**核心洞察**：认知是具身的 (Embodied)、嵌入的 (Embedded)、生成的 (Enacted)、延展的 (Extended)。

**理论来源**：
- SEP Embodied Cognition
- 4E 认知框架
- Gibson 生态心理学
- Merleau-Ponty 身体现象学
- 内感受推理理论

**计算实现**：
```javascript
EmbodiedPredictiveGenerator {
  generateEmbodiedPrediction()     // 生成具身预测
  calculateEmbodiedPredictionError() // 计算具身预测误差
  assess4ECoverage()               // 评估 4E 覆盖度
}
```

**干预应用**：
- 具身自我觉察 (10-15 分钟)
- 主动推理训练 (15-20 分钟)
