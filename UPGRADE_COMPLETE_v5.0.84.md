## 二、理论集成详情

### 2.1 预测加工与具身认知深度整合

**整合框架**:
```
具身预测加工模型 v5.0.84:

传统预测加工:
  感官输入 → 预测误差 → 模型更新 → 新预测

具身预测加工 (增强):
  身体 - 环境耦合 → 主动推理 → 预测误差最小化 → 身体行动 → 新感觉
```

**关键创新点**:
1. **4E 认知框架整合**:
   - Embodied (具身的): 认知依赖于身体
   - Embedded (嵌入的): 认知发生在环境中
   - Extended (延展的): 认知扩展到工具
   - Enactive (生成的): 认知通过耦合产生

2. **主动推理双路径**:
   - 知觉推理：更新内部模型
   - 主动推理：通过行动改变输入

3. **内感受预测核心地位**:
   - 情绪作为身体预测误差的信号
   - 稳态调节作为预测误差最小化

### 2.2 时间意识现象学扩展

**Husserl 时间三重结构**:
```
时间意识流:
  ... → [滞留₁ → 滞留₀ → 原印象 → 前摄₀ → 前摄₁] → ...

特征:
  - 连续性：平滑相位过渡
  - 厚度：现在有时间延展
  - 被动综合：前反思构成统一性
```

**William James 显似现在**:
- 持续时间：6-15 秒
- 功能：使旋律、句子等时间对象感知成为可能
- 特征：同时包含过去和未来

**情绪时间结构**:
1. 情绪滞留：情绪消退后的"余味"
2. 情绪前摄：对即将到来情绪的预期
3. 情绪时间深度：体验的时间延展性

- 频率：每日 2 次 × 5 分钟
- 预期效果：2 周后内感受觉察提升 20%

**本体感受预测训练**:
- 目标：增强身体运动预测
- 方法：闭眼进行动作，预测本体感受
- 频率：每日 1 次 × 10 分钟
- 预期效果：4 周后身体意识提升 25%

**主动推理觉察练习**:
- 目标：识别误差最小化策略
- 方法：记录何时改变世界 vs 改变想法
- 频率：每周反思 3 次
- 预期效果：策略灵活性提升 30%
