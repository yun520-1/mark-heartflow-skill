## 🚀 Quick Start | 快速开始

### ⚠️ Import Gotcha | 导入注意
The standard `from heartflow import ...` fails because `__init__.py` uses `from src.core.heartflow import ...` (relative path). Use `importlib` instead:
标准 `from heartflow import ...` 会失败，因为 `__init__.py` 使用相对路径。请改用 `importlib`：
```python
import importlib.util
spec = importlib.util.spec_from_file_location(
    "heartflow_core",
    "/Users/apple/.hermes/skills/ai/heartflow/src/core/heartflow.py"
)
hf = importlib.util.module_from_spec(spec)
spec.loader.exec_module(hf)
HeartFlow = hf.HeartFlow
process_input = hf.process_input
```

### Basic Usage | 基本用法
```python
# Simple API
result = process_input("帮助别人让我感到快乐")
print(result.decision)

# Full control
engine = HeartFlow()
```

## 🔧 API Reference | API参考

### process_input(user_input, context=None)
Main entry point for all AI assistants.

**Parameters:**
- `user_input` (Any): Text to process | 要处理的文本
- `context` (Dict, optional): Context including challenge_level, skill_level, feedback

**Returns:** DecisionResult with:
- `decision`: Main decision string | 主要决策字符串
- `confidence`: Confidence score (0-1) | 置信度分数
- `reasoning_chain`: Full reasoning breakdown | 完整推理链
- `emotion_analysis`: PAD model results | PAD模型结果
- `consciousness_analysis`: GWT+IIT metrics | GWT+IIT指标
- `flow_state`: Flow state detection | 心流状态检测
- `self_evolution`: Growth metrics | 成长指标
- `ethical_analysis`: TGB evaluation | 真善美评估
