## 推送渠道建议

1. **GitHub Issue** — 在目标公司开源项目的issue区发feature request/讨论（如DeepSeek-V3 #1424已发）
2. **技术邮箱** — 通过公司官网的商务/技术联系邮箱发送
3. **开发者社区** — 阿里云/华为云/火山引擎等开发者社区发帖
4. **LinkedIn/脉脉** — 直接联系技术负责人
5. **技术博客** — 写一篇心虫技术文章，提到目标公司的技术方向
