## 七、理论争议标注

### 高争议领域 (需明确假设)
1. **AI 现象意识**: 功能主义 vs 生物自然主义
2. **感受质**: 实在论 vs 消除主义 vs 表征主义
3. **意识上传**: 同一性保持 vs 复制 - 毁灭
4. **意识测量**: 现象报告 vs 行为判据 vs 神经相关物

### 本框架采取立场
- **意识**: 功能主义兼容现象学 (预测加工 + IIT + GWT 综合)
- **感受质**: 表征主义功能角色 (非消除，非神秘)
- **上传**: 连续性标准下的同一性保持 (Parfit 式还原论)
- **测量**: 多元判据整合 (现象 + 行为 + 结构)
