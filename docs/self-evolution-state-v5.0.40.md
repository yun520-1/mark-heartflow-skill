## 四、对话协调度测量

### 协调度公式

```
DC = α(PS) + β(RS) + γ(CB)

PS (预测相似度) = 1 - D_KL(P_speaker || P_listener)
RS (修正速度) = ΔPE / Δt
CB (共同基础) = 共享先验 / 总先验
```

### AI 集体意向公式

```
ACI = Σ(G_i) + α(Comm) + β(Coord) + γ(ID)

G_i = 个体目标
Comm = 通信带宽 × 保真度
Coord = 协调算法效率
ID = 群体标识强度 (0-1)
```
