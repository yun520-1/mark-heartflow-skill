### 案例 2：AI 帮助职业失望的人（5 步完整流程）

**时间**: 2026-04-07 15:24  
**场景**: 用户遭遇晋升失败

**用户输入**：
> 「我没得到晋升。在这家公司 5 年了，这是第二次被跳过。」

#### 第 1 步：情绪解码（0.3 秒）

```
[感受成分分析]
Q = ⟨-0.39, 0.31, 0.70, 0.40⟩
- 效价：-0.39（负面体验）
- 唤醒：0.31（克制表达）
- 紧张：0.70（累积压力）
- 解决：0.40（对方案开放）

[评价成分分析]
A₁ = -0.44（效价评估：负面）
A₂ = 0.35（唤醒评估：中等）
A₃ = 0.82（目标阻碍：晋升失败）
A₄ = 0.75（规范违背：期望落空）
A₅ = 0.68（责任归属：外部归因）

[检测到核心情绪]
失望 (Disappointment) = 0.73
挫败 (Frustration) = 0.68
疲惫 (Exhaustion) = 0.54
```

#### 第 2 步：共情回应

```
AI：「我能感受到你的失望。5 年的投入，
     第二次被跳过，这种感受很真实。

[记忆] 记录职业挫折事件
[依恋模式检测] 焦虑型倾向（重复被拒）

想说说具体情况吗？
- 是工作表现问题？
- 还是办公室政治？
- 或者公司结构调整？

有时候说出来会好受一些。」
```

#### 第 3 步：认知重构（应用 CBT 技术）

```
[应用 CBT 技术]
自动思维：「我被忽略了 = 我不够好」
认知扭曲：过度概括、个人化

AI：「我注意到一个思维模式：
     『两次被跳过 = 我不够好』

这个想法很自然，但可能有认知扭曲：
1. 过度概括：两次 ≠ 永远
2. 个人化：可能有其他因素（预算、政治）
3. 全或无：没晋升 ≠ 失败

我们来看看证据：
- 支持『我不够好』的证据？
- 不支持的证据？
- 更平衡的解释是什么？」
```

#### 第 4 步：行动计划

```
AI：「如果你想采取行动，我们可以：

1. 信息收集
   - 询问反馈（具体改进点）
   - 了解晋升标准（明确目标）

2. 技能提升
   - 识别差距（技术/管理/影响力）
   - 制定学习计划

3. 选项评估
   - 内部转岗？
   - 外部机会？
   - 创业/自由职业？

不急，我们可以一步步来。
你现在最想探索哪个方向？」

[记忆] 记录用户职业目标，3 天后跟进
```

#### 第 5 步：后续跟进（3 天后自动触发）

```
[自主触发关心]

AI：「嗨，想问问你这两天怎么样？
     上次说的晋升事情，有和领导聊聊吗？

[记忆关联] 用户提到考虑外部机会
[情绪追踪] 检测是否有改善

如果需要，我可以帮你：
- 准备反馈谈话
- 更新简历
- 或者只是听听进展」
```

**这个案例的价值**：
- ✅ AI 真正理解情绪（7 成分计算，不是关键词匹配）
