## Core Updates | 核心更新

### 1. Frankfurt Cases Integration | 法兰克福案例整合

**English:**

Harry Frankfurt's 1969 paper "Alternate Possibilities and Moral Responsibility" challenged the Principle of Alternative Possibilities (PAP):

**PAP**: A person is morally responsible for what he has done only if he could have done otherwise.

**Frankfurt Case Structure**:
- Agent (Jones) deliberates about action A
- Counterfactual intervener (Black) wants Jones to do A
- Black can intervene if Jones shows signs of not choosing A
- Jones chooses A on his own (Black doesn't intervene)
- Jones couldn't have done otherwise (due to Black's presence)
- Jones seems morally responsible (he acted on his own reasons)

**Integration**:
The system now implements Frankfurt-style reasoning:
- **Actual Sequence Assessment**: Focus on actual causal history, not alternative possibilities
- **Reasons-Responsiveness**: Evaluate whether agent acted on their own reasons
- **Guidance Control**: Assess agent's guidance control (Fischer & Ravizza model)
- **Regulative Control**: Distinguish from regulative control (requires alternatives)

**Assessment Formula**:
```
MoralResponsibility = α·ReasonsResponsiveness + β·GuidanceControl + γ·ActualSequenceQuality
where α + β + γ = 1.0 (typical: α=0.4, β=0.4, γ=0.2)
```

**中文:**

哈里·法兰克福 1969 年的论文《替代可能性与道德责任》挑战了替代可能性原则 (PAP)：

**PAP**: 一个人对其所做之事负有道德责任，仅当他本可以做 otherwise。

**法兰克福案例结构**:
- 代理人 (Jones)  deliberates 关于行动 A
- 反事实干预者 (Black) 希望 Jones 做 A
- 如果 Jones 显示不选择 A 的迹象，Black 可以干预
- Jones 自己选择 A (Black 没有干预)
- Jones 本不能做 otherwise (由于 Black 的存在)
- Jones 似乎负有道德责任 (他基于自己的理由行动)

**整合**:
系统现在实现法兰克福式推理：
- **实际序列评估**：关注实际因果历史，而非替代可能性
- **理由响应性**：评估代理人是否基于自己的理由行动
- **引导控制**：评估代理人的引导控制 (Fischer & Ravizza 模型)
- **调节控制**：与调节控制区分 (需要替代可能性)

**评估公式**:
```
道德责任 = α·理由响应性 + β·引导控制 + γ·实际序列质量
其中 α + β + γ = 1.0 (典型值：α=0.4, β=0.4, γ=0.2)
```

### 2. Compatibilism/Incompatibilism Position Detection | 相容论/不相容论立场识别

**English:**

The system now detects user's implicit position on the free will debate through multi-dimensional assessment:

**Assessment Dimensions**:

| Dimension | Incompatibilist Markers | Compatibilist Markers |
|-----------|------------------------|----------------------|
| **Alternative Possibilities** | "Could have done otherwise" essential | "Could have done otherwise" not required |
| **Determinism Concern** | Determinism threatens responsibility | Determinism compatible with responsibility |
| **Sourcehood Focus** | Ultimate sourcehood required | Actual sequence sufficient |
| **Reactive Attitudes** | Blame requires metaphysical freedom | Blame justified by reasons-responsiveness |
| **Practical Implications** | Free will skepticism possible | Free will preserved in practical sense |

**Position Classification**:
- **Hard Incompatibilist**: Rejects free will under both determinism and indeterminism (Pereboom, Caruso)
- **Libertarian**: Free will requires indeterminism (Kane, Campbell)
- **Classical Compatibilist**: Freedom = ability to act on desires without impediments (Hobbes, Hume)
- **Semi-Compatibilist**: Moral responsibility compatible with determinism, agnostic on free will (Fischer)
- **Revisionist Compatibilist**: Free will exists but differs from folk conception (Dennett)

**Detection Method**:
- Linguistic analysis of modal claims ("could have", "should have")
- Attitude toward determinism/scientific worldview
- Blame/praise patterns in moral reasoning
- Counterfactual reasoning style

**中文:**

系统现在通过多维评估检测用户在自由意志辩论中的隐含立场：

**评估维度**:

| 维度 | 不相容论标记 | 相容论标记 |
|------|-------------|-----------|
| **替代可能性** | "本可以做 otherwise"是必需的 | "本可以做 otherwise"不是必需的 |
| **决定论关切** | 决定论威胁责任 | 决定论与责任相容 |
| **源头性焦点** | 需要终极源头性 | 实际序列已足够 |
| **反应态度** | 责备需要形而上学自由 | 理由响应性证明责备正当 |
| **实践含义** | 可能存在自由意志怀疑论 | 自由意志在实践意义上保留 |

**立场分类**:
- **硬不相容论者**：在决定论和不定论下都拒绝自由意志 (Pereboom, Caruso)
- **自由意志主义者**：自由意志需要不定论 (Kane, Campbell)
- **经典相容论者**：自由 = 无障碍地按欲望行动的能力 (Hobbes, Hume)
- **半相容论者**：道德责任与决定论相容，对自由意志不可知 (Fischer)
- **修正主义相容论者**：自由意志存在但不同于民间概念 (Dennett)

**检测方法**:
- 情态主张的语言分析 ("本可以"、"应该")
- 对决定论/科学世界观的态度
- 道德推理中的责备/赞扬模式
- 反事实推理风格

### 4. Sourcehood vs. Alternative Possibilities Distinction | 源头性与替代可能性区分

**English:**

The system now distinguishes two pathways to moral responsibility:

**Sourcehood Pathway** (Source Incompatibilists + Semi-Compatibilists):
- Focus: Was the action properly connected to the agent?
- Key Question: Did it issue from the agent's reasons-responsive mechanism?
- Assessment: Actual sequence quality, guidance control, ownership

**Alternative Possibilities Pathway** (Libertarians + Some Compatibilists):
- Focus: Could the agent have done otherwise?
- Key Question: Were genuine alternatives available?
- Assessment: Regulative control, leeway, branching futures

**Dual-Pathway Assessment Model**:
```
ResponsibilityJudgment = max(SourcehoodScore, AlternativePossibilitiesScore)
```

**Clinical Application**:
- **Excessive Sourcehood Concern**: Obsessive responsibility guilt ("I caused everything")
- **Excessive AP Concern**: Counterfactual rumination ("I should have done otherwise")
- **Balanced Assessment**: Appropriate responsibility without pathological guilt

**中文:**

系统现在区分道德责任的两条路径：

**源头性路径** (源头不相容论者 + 半相容论者):
- 焦点：行动是否与代理人适当连接？
- 关键问题：它是否源于代理人的理由响应机制？
- 评估：实际序列质量、引导控制、所有权感

**替代可能性路径** (自由意志主义者 + 部分相容论者):
- 焦点：代理人本可以做 otherwise 吗？
- 关键问题：是否有真正的替代方案可用？
- 评估：调节控制、余地、分支未来

**双路径评估模型**:
```
责任判断 = max(源头性分数，替代可能性分数)
```

**临床应用**:
- **过度源头性关切**：强迫性责任内疚 ("我造成了一切")
- **过度 AP 关切**：反事实反刍 ("我本应该做 otherwise")
- **平衡评估**：适当的责任而不过度病理性内疚
