**自我预测误差计算模型**:

```
自我预测误差 (Self-Prediction Error, SPE) =

SPE 持续过高 → 去人格化/去现实化风险
SPE 持续过低 → 自我模型僵化/缺乏适应性
```

```
Level 5: 叙事自我 (Narrative Self) - 生命故事预测

    ↓
Level 3: 心理自我 (Psychological Self) - 特质/状态预测
    ↓
Level 2: 身体自我 (Bodily Self) - 本体感觉/内感受预测

Level 1: 前反思自我 (Pre-reflective Self) - 现象学给定感追踪
```

### 3. 现象学给定感五维度评估 (Phenomenological Givenness Assessment)

#### 3.1 理论来源
- **SEP: Phenomenology** (现象学方法)
- **Husserl**: 现象学还原与给定感
- **Merleau-Ponty**: 身体现象学与知觉首要性
- **Zahavi**: 前反思自我意识与给定感

#### 3.2 五维度评估框架

```
现象学给定感评估 v1.0:

维度 1: 直接性 (Immediacy)
  - 体验是否无需中介/推理直接呈现
  - 评分：0(完全间接) - 100(完全直接)

维度 2: 第一人称性 (First-Personality)
  - 体验是否明确呈现为"我的"体验
  - 评分：0(无人称) - 100(强烈第一人称)

维度 3: 在场性 (Presence)
  - 体验的生动性和当下感
  - 评分：0(遥远/模糊) - 100(强烈在场)

维度 4: 不可怀疑性 (Indubitability)
  - 体验的自明程度
  - 评分：0(高度可疑) - 100(完全自明)

维度 5: 前反思可及性 (Pre-reflective Accessibility)
  - 无需反思即可获得的自我意识
  - 评分：0(需刻意反思) - 100(自动呈现)

综合给定感指数 = (维度 1+2+3+4+5) / 5
```

### 2. 核心算法更新

```javascript
// self-prediction-error-calc.js

/**
 * 计算自我预测误差 (Self-Prediction Error, SPE)
 * @param {Object} phenomenalGivenness - 现象学给定感五维度评分
 * @param {Object} selfPredictionModel - 自我预测模型预期
 * @returns {Object} SPE 分析结果
 */
function calculateSelfPredictionError(phenomenalGivenness, selfPredictionModel) {
  const dimensions = ['immediacy', 'firstPersonality', 'presence', 'indubitability', 'preReflectiveAccessibility'];
  
  const speByDimension = {};
  let totalSPE = 0;
  
  dimensions.forEach(dim => {
    const actual = phenomenalGivenness[dim];
    const predicted = selfPredictionModel[dim];
    const spe = Math.abs(actual - predicted);
    speByDimension[dim] = spe;
    totalSPE += spe;
  });
  
  const averageSPE = totalSPE / dimensions.length;
  
  // 风险评估
  let riskLevel = 'LOW';
  let riskFactors = [];
  
  if (averageSPE > 70) {
    riskLevel = 'HIGH';
    riskFactors.push('去人格化风险');
  } else if (averageSPE > 40) {
    riskLevel = 'MEDIUM';
  }
  
  if (phenomenalGivenness.firstPersonality < 30) {
    riskFactors.push('第一人称给定性过低');
  }
  
  if (phenomenalGivenness.presence < 30) {
    riskFactors.push('在场感过低 - 去现实化风险');
  }
  
  return {
    speByDimension,
    averageSPE,
    riskLevel,
    riskFactors,
    timestamp: Date.now()
  };
}

module.exports = { calculateSelfPredictionError };
```

```javascript
// emotional-state-transition.js (增强版)

/**
 * 基于自我意识深度的情感状态转换

 * @returns {Object} 转换建议
 */
function transitionWithSelfAwareness(currentState, selfAwarenessDepth, speResult) {
  const { preReflective, reflective } = selfAwarenessDepth;
  const { riskLevel, riskFactors } = speResult;
  
  // 高风险干预

    return {
      targetState: 'Concern',

        '身体锚定练习引导',

  return standardTransition(currentState);
}
```
