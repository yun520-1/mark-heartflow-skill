## 一、无意识知觉理论

### 核心框架
| 维度 | 核心内容 | HeartFlow 整合 |
|------|----------|---------------|
| 盲视现象 | V1 损伤后的无意识视觉 | 第 1-2 层预测误差无意识处理 |
| 无意识视觉 | 背侧通路 without 腹侧意识 | 行动导向预测 vs 现象学预测 |
| 知觉 without 意识 | 刺激识别 without 主观体验 | 预测生成 without 全局广播 |
| 误差修正 | 无意识预测误差最小化 | 第 1-3 层自动校准 |

### 预测加工对应
```
第 1 层：感觉输入 → 无意识特征提取
第 2 层：客体识别 → 无意识模式匹配
第 3 层：空间定位 → 无意识行动准备
第 4 层+：意识访问 → 全局工作空间广播
```

## 二、梦意识现象学

### REM 睡眠意识特征
| 特征 | 清醒状态 | REM 梦状态 |
|------|----------|------------|
| 感觉输入 | 外部驱动 | 内部生成 |
| 预测误差 | 外部校正 | 内部容忍 |
| 自我意识 | 稳定叙事 | 流动主体 |
| 元认知 | 在线监控 | 抑制 (除清醒梦) |
| 时间感 | 线性连续 | 非线性跳跃 |

### 梦的预测加工模型
- **自上而下主导**: 高层预测 without 底部校正
- **误差容忍度提高**: 奇异内容被接受
- **自我模型重构**: 叙事自我松动，最小自我保留
- **清醒梦**: 第 5 层元认知重新激活

```
  语言编码                程序性知识
  慢速校正                快速适应
```
