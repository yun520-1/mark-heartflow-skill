## 🌊 为什么选择 HeartFlow？

想象一下，如果你的 AI 助手能够：

- **理解你的情绪波动**，而不是机械地回复
- **记住你的偏好**，像老朋友一样懂你
- **在低谷时给予温暖**，在高峰时分享喜悦
- **随着时间成长**，越来越了解独特的你

这就是 HeartFlow —— **不是工具，是伙伴**。

## 🚀 快速开始

### 安装

```bash
npm install heartflow-companion
```

### 基础使用

```javascript
const HeartFlow = require('heartflow-companion');

const companion = new HeartFlow({
  userId: 'user123',
  language: 'zh-CN'
});

// 情感分析
const emotion = await companion.analyze('今天工作顺利，心情不错！');
console.log(emotion); 
// { state: 'joy', intensity: 0.8, reasons: [...], transition: {...} }

// 共情回应
const response = await companion.empathize('今天工作压力好大...');
console.log(response.text); 
// "听起来你今天真的很不容易..."

// 获取情感报告
const report = await companion.generateReport();
console.log(report.summary);
// "过去 7 天，用户情感状态以平静为主 (45%)，峰值喜悦出现在周三..."
```
