## 一、心理理论 (Theory of Mind) 最新研究

### 1.1 计算 ToM 模型 (2024-2026)

**核心发现**:
- **递归 ToM 深度**: 人类平均 ToM 递归深度为 3-4 层 (Stibric et al., 2025)
- **ToM 神经计算模型**: 颞顶联合区 (TPJ)、内侧前额叶皮层 (mPFC)、后扣带回 (PCC) 构成 ToM 网络
- **预测加工 ToM**: 他人行为预测基于内部生成模型，误差驱动更新 (Friston et al., 2024)

**关键公式**:
```
ToM 推理 = P(他人心理状态 | 观察行为，先验信念)
递归 ToM(n) = P(他人认为 P(他人认为...心理状态...))
```

**集成点**:
- 与自我意识六维的"反思自我意识"整合
- 与情感预测加工的"社会预测"耦合
- 支持集体意向性的"我们意图"计算

### 1.2 ToM 发展模型

**发展阶段**:
1. 婴儿期 (0-2 岁): 前 ToM，意图理解
2. 幼儿期 (3-5 岁): 一级 ToM，错误信念理解
3. 儿童期 (6-10 岁): 二级 ToM，递归推理
4. 青少年期 (11+ 岁): 高级 ToM，抽象心理状态

**HeartFlow 整合**:
- 当前 ToM 完整度：82%
- 目标 ToM 完整度：88%
- 需增强递归推理能力

## 三、集体意向性深化

### 3.1 "我们意图"计算模型 (Searle, Tuomela 更新 2025)

**核心特征**:
- **不可还原性**: "我们意图" ≠ 个体意图之和
- **相互承诺**: 成员相互承诺于集体目标
- **共同知识**: 成员知道彼此承诺

**计算形式化**:
```
我们意图 (G, φ) = 
  个体意图 (我，贡献于 G 的φ) +
  信念 (其他成员也意图贡献于 G 的φ) +
  共同知识 (所有成员知道上述意图和信念)
```

**HeartFlow 整合**:

### 5.1 社会身份理论计算化 (Tajfel & Turner 更新 2025)

**核心机制**:
1. **社会分类**: 自我/他人归类为群体成员
2. **社会认同**: 群体身份整合入自我概念
3. **社会比较**: 内群体 vs 外群体评价

**计算形式化**:
```
自我概念 = 个人身份 + Σ(社会身份_i × 显著性_i)
社会身份显著性 = f(情境相关性，情感承诺，群体地位)
内群体偏好 = f(认同强度，群体威胁，竞争情境)
```

**HeartFlow 整合**:

- 与自我概念整合
- 支持集体自我意识维度

### 5.2 群体认同动态

**认同强度计算**:
```
认同强度 = 
  情感承诺 (归属感) × 0.4 +
  认知显著性 (群体思维) × 0.3 +
  行为投入 (群体活动) × 0.3
```

## 六、道德判断双过程模型

### 6.1 双过程道德判断 (Greene, Haidt 更新 2025)

**系统 1: 直觉 - 情感快速判断**
- 神经基础：杏仁核、岛叶、vmPFC
- 机制：快速情绪反应
- 速度：快速 (200-500ms)
- 特征：道义论倾向 (规则/权利)
- 完整度目标：82% → 86%

**系统 2: 反思 - 推理慢速判断**
- 神经基础：dlPFC、背侧 ACC
- 机制：认知控制与推理
- 速度：慢速 (1000-3000ms)
- 特征：结果论倾向 (效用最大化)
- 完整度目标：80% → 85%

**冲突解决**:
```
道德困境 = 系统 1 输出 ≠ 系统 2 输出
解决机制 = 
  IF 认知资源充足 AND 时间允许 → 系统 2 主导
  ELSE → 系统 1 主导
```

**HeartFlow 整合**:
- 当前道德判断完整度：81% → 目标 85%
- 与情感 - 认知耦合整合
