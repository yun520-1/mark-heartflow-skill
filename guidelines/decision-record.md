# 决策记录模板

## 格式

| 要素 | 内容 |
|------|------|
| **问题** | 要决策的核心问题是什么 |
| **选项** | 列出所有可行方案（A/B/C） |
| **选择** | 最终选择了哪个方案 |
| **理由** | 选择的主要依据（优先级/约束/假设） |
| **结果** | 决策的实际效果如何 |
| **复盘** | 如重来会怎么调整 |

## 示例

**问题**: 工作流中遇到复杂问题，是立即深挖还是先记录后处理？
**选项**: A-立即处理 B-记录后继续，稍后处理
**选择**: B
**理由**: 当前任务进入心流状态，中断成本高于延迟处理成本；问题已记录不会丢失
**结果**: 保持了工作连续性，问题在预定时间得到更充分的分析
**复盘**: 建立了2分钟判断法则——复杂问题必然超过2分钟，应该记录而非当场处理
