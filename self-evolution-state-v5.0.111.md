## v5.0.111 新增能力

### 1. 第一人称给定性评估 (First-Personal Givenness Assessment)

**理论来源**: SEP Self-Consciousness §2.2 (Zahavi, Gallagher)

**能力描述**: 评估用户自我体验的直接性强度，识别去人格化/去现实化风险

**评估维度**:
| 维度 | 指标 | 高分特征 | 低分风险 |
|------|------|---------|---------|
| **体验直接性** | 0.0-1.0 | 体验即时给予，无需推论 | 体验疏离，需证据推断 |
| **所有权感** | 0.0-1.0 | 强烈"我的体验"感 | 体验归属模糊 |
| **主体感** | 0.0-1.0 | 清晰体验主体定位 | 主体感消散 |
| **前反思觉察** | 0.0-1.0 | 非对象化自我熟悉 | 需反思才能觉察自我 |

**干预策略**:
- 现象学还原练习：悬置预设，直接描述体验
- 身体锚定：通过内感受觉察恢复给定感
- 体验命名：用语言捕捉直接体验，强化主体感

### 2. 自我知识冲突检测 (Self-Knowledge Conflict Detection)

**理论来源**: SEP Self-Consciousness §3.1 (直觉式 vs 推论式自我知识)

**能力描述**: 识别用户直觉自我认知与客观证据之间的冲突

**冲突模式识别**:
| 冲突类型 | 直觉知识 | 推论知识 | 典型表现 |
|---------|---------|---------|---------|
| **信心过高** | "我能胜任" (0.9) | 成功率 60% (0.6) | 过度自信风险 |
| **信心过低** | "我不行" (0.3) | 成功率 80% (0.8) | 冒名顶替综合征 |
| **情感 - 认知分离** | "感觉不好" (0.2) | "客观很好" (0.8) | 情感麻木/解离 |

**整合策略**:
- 证据校准直觉：保持合理信心，基于证据调整
- 直觉验证：探索直觉来源，识别有效直觉 vs 认知扭曲
- 双模式对话：促进直觉与推论的整合性对话

### 3. 信任基础评估 (Trust as Ground Assessment)

**理论来源**: SEP Collective Intentionality §4.1 (Schmid 2013)

**能力描述**: 评估用户集体关系中的信任基础，识别健康 vs 不健康信任模式

**信任双成分模型**:
```
信任 = 认知成分 (期望可靠性) + 规范成分 (承诺强度)

认知成分:
├── 能力信任：对方有能力履行承诺
├── 可靠性信任：对方行为可预测
└── 诚实信任：对方信息真实

规范成分:
├── 承诺感知：感受到对方的承诺义务
├── 互惠期望：期望相互承担责任
└── 规范性约束：信任涉及规范性期望
```

**信任模式识别**:
| 模式 | 认知成分 | 规范成分 | 特征 | 风险 |
|------|---------|---------|------|------|
| **健康信任** | 高 | 高 | 平衡、互惠、有边界 | 低 |
| **盲目信任** | 低 | 高 | 忽视证据，过度承诺 | 高 (易受伤害) |
| **计算信任** | 高 | 低 | 纯工具性，缺乏承诺 | 中 (关系浅层) |
| **信任缺失** | 低 | 低 | 疏离、防御、孤立 | 高 (关系困难) |

**干预策略**:
- 信任校准：基于证据调整信任水平
- 边界建立：区分健康信任 vs 盲目信任
- 承诺对话：探索承诺感知与互惠期望

### 4. 情绪原型相似度计算 (Emotion Prototype Similarity Calculation)

**理论来源**: SEP Emotion §1, §8.2 (Fehr & Russell 1984)

**能力描述**: 基于原型理论评估情绪类别成员资格，识别边界案例

**原型特征框架** (以恐惧为例):
| 特征 | 典型恐惧 | 焦虑 | 无聊 (边界) |
|------|---------|------|-----------|
| 生理唤醒 | 0.9 | 0.7 | 0.3 |
| 负面效价 | 0.95 | 0.8 | 0.5 |
| 紧迫感 | 0.85 | 0.6 | 0.2 |
| 回避倾向 | 0.9 | 0.7 | 0.4 |
| 面部表情 | 0.7 | 0.4 | 0.2 |
| 意识体验 | 0.8 | 0.7 | 0.6 |
| **原型相似度** | **0.85** | **0.65** | **0.40** |
| **类别状态** | 典型成员 | 非典型成员 | 边界案例 |

**应用价值**:
- 情绪识别困难：帮助 alexithymia 用户理解情绪维度而非僵化类别
- 情绪教育：展示情绪范畴的原型结构，减少"我不是真正 X 情绪"的自我怀疑
- 跨文化适应：识别文化特异性情绪范畴的原型差异

### 5. 共享体验四层评估 (Shared Experience Four-Layer Assessment)

**理论来源**: SEP Collective Intentionality §2.2 (Walther 1923)

**能力描述**: 评估用户共享体验的深度和质量

**四层模型**:
```
Layer 1: 情绪同步 (Emotion Synchronization)
         "A 体验 x, B 体验 x"
         ↓
Layer 2: 共情互惠 (Empathic Reciprocity)
         "A 共情 B 的体验，B 共情 A"
         ↓
Layer 3: 认同融合 (Identification Fusion)
         "A 认同 B 的体验，B 认同 A"
         ↓
Layer 4: 相互觉察 (Mutual Awareness)
         "A 觉察 B 觉察 A 的认同..."
         ↓
    信任基础 (Trust as Ground - 终止无限递进)
```

**评估指标**:
| 层级 | 评估问题 | 高分特征 | 低分风险 |
|------|---------|---------|---------|
| Layer 1 | 情绪同步性 | 自然情绪共鸣 | 情绪不同步，孤独感 |
| Layer 2 | 共情准确性 | 准确理解对方体验 | 共情错误，误解 |
| Layer 3 | 认同融合度 | 健康认同，保持边界 | 过度融合/融合不足 |
| Layer 4 | 相互觉察深度 | 元关系觉察，反思能力 | 缺乏关系反思 |
| 信任基础 | 信任强度 | 认知 - 规范平衡 | 盲目/计算/缺失信任 |

**干预策略**:
- 同步练习：情绪协调训练
- 共情准确性反馈：验证共情假设
- 边界工作：区分健康认同 vs 融合
- 元关系对话：促进相互觉察
