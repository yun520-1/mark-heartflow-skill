## New Theory Modules | 新增理论模块

### 1. Three-Tradition Emotion Framework | 三大传统情绪框架

**English:**
Integrates SEP's three traditions in emotion theory:
- **Feeling Tradition**: James-Lange → Modern psychological constructionism
- **Evaluative Tradition**: Cognitive appraisal → Fittingness assessment
- **Motivational Tradition**: Action tendency → Behavioral activation

**中文:**
整合 SEP 情绪理论三大传统：
- **感受传统**: 詹姆斯 - 兰格 → 现代心理建构主义
- **评价传统**: 认知评估 → 恰当性评估
- **动机传统**: 行动倾向 → 行为激活

### 2. Dual-Layer Self-Consciousness | 双层自我意识

**English:**
Based on SEP Self-Consciousness (2025):
- **Pre-reflective**: Immediate, non-objectifying, first-personal givenness
- **Reflective**: Mediated, objectifying, conceptual self-knowledge
- **Integration**: Self-alienation detection (depersonalization/derealization)

**中文:**
基于 SEP 自我意识 (2025)：
- **前反思**: 直接的、非对象化的、第一人称给定性
- **反思**: 中介的、对象化的、概念自我知识
- **整合**: 自我异化检测 (去人格化/去现实化)

### 3. Walther Four-Layer Model | Walther 四层模型

**English:**
Shared experience requires four layers:
1. Co-experience (both experience x)
2. Empathic awareness (empathize with other's experience)
3. Identification (identify with other's experience)
4. Mutual awareness (aware of layers 1-3)

**中文:**
共享体验需要四层：
1. 共体验 (都体验 x)
2. 共情觉察 (共情对方体验)
3. 认同 (认同对方体验)
4. 相互觉察 (觉察 1-3 层)

### 4. Scheler Collective Emotion | Scheler 集体情绪

**English:**
Three types of collective emotion:
- **Directly shared**: Intimate co-feeling (e.g., parents' grief)
- **Atmospheric**: Crowd emotion, contagion-like
- **Intentional**: Shared commitment to cause (we-intention based)

**中文:**
三种集体情绪类型：
- **直接共享**: 亲密共感 (如父母悲伤)
- **氛围性**: 群体情绪、类似传染
- **意向性**: 对事业的共同承诺 (基于我们 - 意向)
