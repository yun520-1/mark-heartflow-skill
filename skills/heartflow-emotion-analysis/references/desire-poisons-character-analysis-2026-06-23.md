# 欲望+三毒人物分析实战记录（2026-06-23）

## 背景
用户要求用心虫分析《卡徒》人物的七情六欲和贪嗔痴。核心教训：**人是被欲望驱动的，不是被配对逻辑驱动的。**

## 关键纠正
1. "荣铭已经结婚了，而且有12岁小孩" → 不要默认所有异性角色之间都有浪漫可能性
2. "不要我告诉你爱情，你就把所有人进行配对，他们是人，不是猪" → 七情六欲是用来理解个体命运的，不是用来产生关系图谱的

## 正确的人物分析顺序

### 第一步：欲望分析（desire-cognition.js）
```
hf.dispatch('desireCognition.analyzeDesires', {name, traits})
→ 输出9种欲望维度的评分
→ 核心欲望决定命运方向
```

### 第二步：情感分析（desire-cognition.js）
```
hf.dispatch('desireCognition.analyzeSevenEmotions', {name, traits})
hf.dispatch('desireCognition.analyzeValenceArousal', {name, traits})
→ 七情分布 + VA情感空间
```

### 第三步：驱动力分析（desire-cognition.js）
```
hf.dispatch('desireCognition.analyzeDriveSatisfaction', {name, traits})
→ 7驱动力满意度，最低驱力揭示核心焦虑
```

### 第四步：三毒分析（three-poisons.js）— 可选
当用户问"人性弱点"、"为什么堕落"时使用
```
tp.analyzeThreePoisons({name, greed, hatred, delusion})
tp.analyzePoisonsDrivenFate({name})
```

### 第五步：关系分析（仅当用户明确要求）
```
hf.dispatch('desireCognition.analyzeDesireInteraction', personA, personB)
```
**不要默认做这一步。** 只有用户明确说"XX和YY的关系如何"或"分析XX和YY"才做。

## 七种关系类型（非只有浪漫）

| 关系类型 | 主导情感 | 示例（卡徒） |
|---------|---------|-------------|
| 师徒/传承 | 爱（关怀型） | 西泽→陈暮 |
| 对手/宿敌 | 怒/惧 | 唐含沛→陈暮 |
| 战友/同伴 | 爱（信任型） | 陈暮→维阿 |
| 上下级/权力 | 恶（尊重/厌恶） | 唐含沛→谈雨玟 |
| 家族/血缘 | 爱（义务型） | 陈暮无血缘关系 |
| 浪漫（仅当书中有明确信号） | 爱（依恋型）+欲 | 陈暮→苏流澈柔（书中明确） |
| 无关 | 无 | 荣铭→许嘉（无信号不分析） |

## 卡徒人物欲望+三毒分析结果

### 陈暮
- **欲望驱动**：生存欲(0.90) + 自由欲(0.90) + 社交欲(0.80)
- **PADCN**：P=-0.05 A=0.43 D=0.08 C=0.17 N=0.10
- **三毒**：贪6.3 嗔5.5 痴4.5 | 总毒5.4(中度)
- **命运**：常人型，生存→成就驱动，多重欲望交织
- **关系**：和苏流澈柔是书中明确描写的浪漫关系（她给他做早餐，他觉得很不错）

### 唐含沛
- **欲望驱动**：权力欲(0.80) + 成就欲(0.80)
- **PADCN**：P=-0.05 A=0.48 D=0.19 C=0.25 N=0.11
- **三毒**：贪7.3 嗔7.0 痴6.3 | 总毒6.8(中重度)
- **命运**：挣扎型，权力异化人际关系，最终孤独
- **关系**：和谈雨玟是权力交换关系，非浪漫

### 维阿
- **欲望驱动**：生存欲(0.90) + 自由欲(0.90)
- **三毒**：贪3.3 嗔5.5 痴4.8 | 总毒4.5(轻度)
- **命运**：常人型，自由驱动，归宿风险

### 苏流澈柔
- **欲望驱动**：社交欲(0.80)
- **三毒**：贪3.0 嗔4.0 痴4.0 | 总毒3.7(低毒)
- **命运**：健康型，归属驱动

### 肖波
- **欲望驱动**：自由欲(0.90) + 性欲(0.80)
- **三毒**：贪5.8 嗔5.0 痴4.8 | 总毒5.2(轻度)
- **命运**：常人型，自由驱动

### 谈雨玟
- **欲望驱动**：社交欲(0.80) + 权力欲(0.80)
- **三毒**：贪7.3 嗔6.0 痴5.8 | 总毒6.3(中重度)
- **命运**：挣扎型，权力+归属交织

### 裘珊玉
- **欲望驱动**：自由欲(0.90) + 权力欲(0.80) + 物欲(0.70)
- **三毒**：贪7.8 嗔7.3 痴7.3 | 总毒7.4(重度)
- **命运**：**悲剧型**，三毒全燃——贪驱动永不满足，嗔驱动报复心极强，痴驱动完全看不清自己
