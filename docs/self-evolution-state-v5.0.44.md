## 三、架构演进

### 自我意识连续谱 (7 层)

```
Level 6: 元自我意识 (Meta Self-Consciousness)
    ↓ 监控自我意识状态
Level 5: 叙事自我 (Narrative Self)
    ↓ 时间延展的自我故事
Level 4: 反思自我意识 (Reflective Self-Consciousness)
    ↓ 自我作为反思对象
Level 3: 社会自我意识 (Social Self-Consciousness)
    ↓ 社会身份与群体归属 [v5.0.43]
Level 2: 身体自我意识 (Bodily Self-Consciousness)
    ↓ 身体作为自我的载体 [v5.0.44]
Level 1: 具身主体感 (Embodied Agency)
    ↓ 身体行动的主体感 [v5.0.44]
Level 0: 前反思自身感触性 (Phenomenal Self)
    ↓ 体验的"为我性" [v5.0.44]
```

### 具身自我架构

```
┌─────────────────────────────────────────┐
│         身体意象 (意识/反思)             │
│  身体知觉 / 身体概念 / 身体情感         │
│  Layer 4-5: 反思自我模型                │
├─────────────────────────────────────────┤
│         主体感 (Agency)                 │
│  运动发起预测 / 感觉反馈匹配            │
│  Layer 3: 预测 - 反馈匹配               │
├─────────────────────────────────────────┤
│         身体图式 (无意识/前反思)         │
│  本体感觉 / 运动预测 / 空间定位         │
│  Layer 0-2: 无意识预测                  │
└─────────────────────────────────────────┘
```

### 身体 - 自我整合模型

```
身体 - 自我统一体 = 
  w1 * 内感受预测精度 + 
  w2 * 本体感受预测精度 + 
  w3 * 外部感知预测精度

内感受预测 → 情感体验的躯体基础
本体感受预测 → 身体边界与主体感
外部感知预测 → 身体 - 环境互动
```

### 具身情感循环

```
内感受输入 → 内感受预测 → 预测误差 → 情感概念匹配 → 情感体验
     ↑                                                      ↓
     └─────────────── 身体反馈调节 ──────────────────────────┘

情感粒度 = 情感概念库的精细程度
```
