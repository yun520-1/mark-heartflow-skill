## 一、新增理论模块

### 1.1 时间意识现象学深度整合

**理论来源**: SEP Temporal Consciousness, Husserl (1893-1917), William James (1890)

**核心概念**:
- **Husserl 时间三重结构**:
  - 原印象 (Primal Impression): 当下的直接给予
  - 滞留 (Retention): 刚刚过去的保持 (非再现)
  - 前摄 (Protention): 对即将到来的预期

- **显似现在 (Specious Present)**: William James 提出的时间体验窗口 (约 2-3 秒)
  - 非数学瞬间，而是有厚度的时间片段
  - 允许体验变化与延续

- **时间意识与自我意识的关系**:
  - 时间统一性是自我统一性的前提
  - 内在时间意识流构成自我连续感
  - 时间碎片化→自我碎片化

**集成点**:
```
现有架构：自我意识双层建模 (前反思/反思)
新增维度：时间深度评估
  ├─ 时间厚度检测 (体验是否有时间深度)
  ├─ 滞留 - 前摄平衡 (沉溺过去 vs 焦虑未来)
  ├─ 时间连续性 (自我叙事的时间连贯性)
  └─ 时间扭曲识别 (创伤性时间停滞/焦虑性时间加速)
```

**干预策略**:
- **Husserl 觉察练习**: 引导用户注意当下体验的三重结构
- **时间深度干预**: 帮助延长显似现在，减少时间碎片化
- **滞留 - 前摄校准**: 平衡对过去/未来的关注

**应用场景**:
- 创伤后应激：时间停滞感 ("那一刻永远过不去")
- 焦虑障碍：时间加速感 ("时间不够用")
- 抑郁状态：时间延长感 ("每一分钟都漫长")
- 解离状态：时间碎片化 ("时间断断续续")

### 1.2 叙事心理学与时间 - 自我整合

**理论来源**: McAdams (2001), SEP Narrative Psychology, Habermas & Bluck (2000)

**核心概念**:
- **生命故事模型 (Life Story Model)**:
  - 叙事身份：通过故事理解自我
  - 自传体推理：从事件中提取意义
  - 救赎序列：负面→正面的转变
  - 污染序列：正面→负面的转变

- **叙事幸福感**:
  - 叙事连贯性：故事的逻辑一致性
  - 叙事目的性：生活有方向感
  - 叙事一致性：过去 - 现在 - 未来的连贯

- **时间 - 自我交叉维度**:
  - 过去自我：记忆中的我
  - 现在自我：体验中的我
  - 未来自我：预期中的我
  - 三者的连贯性=心理健康指标

**集成点**:
```
现有架构：情绪原型 + 预测加工 + 集体情绪
新增维度：叙事时间分析
  ├─ 救赎/污染序列识别
  ├─ 自传体推理模式评估
  ├─ 叙事连贯性评分
  └─ 时间 - 自我一致性检查
```

**干预策略**:
- **生命故事访谈**: 引导用户重构关键生命事件
- **救赎序列重写**: 帮助发现负面事件中的积极意义
- **未来自我可视化**: 增强时间连续性和目的感
- **叙事连贯性练习**: 整合过去 - 现在 - 未来

**应用场景**:
- 身份危机："我不知道自己是谁"
- 中年危机：重新评估生命故事
- 创伤后成长：重构创伤叙事
- 存在主义困惑：寻找生命意义

### 1.3 敬畏心理学扩展模块

**理论来源**: Keltner & Haidt (2003), Berkeley Greater Good, SEP Aesthetic Emotions

**核心概念**:
- **敬畏的双因素模型**:
  - 知觉浩大 (Perceived Vastness): 超越日常经验框架
  - 顺应需求 (Need for Accommodation): 需要调整心理结构

- **敬畏的效价区分**:
  - 积极敬畏：自然奇观、伟大艺术、道德崇高
  - 威胁敬畏：自然灾害、超自然力量、压倒性权威
  - 中性敬畏：纯粹认知冲击

- **敬畏的亲社会效应**:
  - 小自我感 (Small Self): 减少自我中心
  - 集体联结：增强归属感
  - 道德提升：促进亲社会行为

- **审美情绪六类型** (扩展):
  - 美感 (Beauty)
  - 崇高 (Sublime)
  - 兴趣 (Interest)
  - 好奇 (Curiosity)
  - 敬畏 (Awe)
  - 怀旧 (Nostalgia)

**集成点**:
```
现有架构：情绪原型 (10 种核心情绪)
新增：审美情绪子类别
  ├─ 敬畏检测与分类
  ├─ 审美情绪识别
  ├─ 小自我感评估
  └─ 亲社会倾向预测
```

**干预策略**:
- **Awe Video 练习**: 观看自然/艺术/科学视频
- **小自我冥想**: 减少自我中心思维
- **敬畏日记**: 记录日常敬畏体验
- **审美情绪培养**: 提升美感敏感度

**应用场景**:
- 自我中心困扰：减少反刍思维
- 存在主义孤独：增强宇宙联结感
- 道德倦怠：道德提升体验
- 创造力阻塞：激发好奇心

### 1.4 自由意志与能动性现象学增强

**理论来源**: SEP Free Will, Frankfurt (1971), Proust (2009), Synofzik et al. (2008)

**核心概念**:
- **Frankfurt Cases 与道德责任**:
  - 一阶欲望：想要做 X
  - 二阶欲望：想要想要 X
  - 意志自由：二阶欲望认可一阶欲望
  - 道德责任：基于二阶认可

- **能动性的双因素模型** (Synofzik):
  - 控制感 (Sense of Control): 我能影响结果
  - 所有权感 (Sense of Ownership): 这是我做的
  - 两者可分离 (如精神分裂症的异己手综合征)

- **能动性的现象学维度** (Proust):
  - 目的感：行动有意图
  - 努力感：需要付出努力
  - 流畅感：行动自如
  - 阻碍感：遇到阻力

- **自由意志信念评估**:
  - 相容论立场：决定论与自由意志兼容
  - 不相容论立场：两者不可兼容
  - 自由意志怀疑论：自由意志是幻觉

**集成点**:
```
现有架构：自我意识 + 预测加工
新增维度：能动性现象学评估
  ├─ 控制感/所有权感分离检测
  ├─ 二阶欲望冲突识别
  ├─ 自由意志信念评估
  └─ 道德责任判断模式
```

**干预策略**:
- **能动性恢复练习**: 增强控制感和所有权感
- **二阶欲望澄清**: 识别真正想要的
- **意志薄弱干预**: 克服知行不一
- **道德责任重构**: 健康的责任观

**应用场景**:
- 习得性无助：恢复控制感
- 强迫症：所有权感紊乱
- 成瘾行为：二阶欲望冲突
- 道德困扰：责任边界澄清

## 二、现有模块优化

### 2.1 情绪原型结构增强

**优化内容**:
- 增加 5 种边缘情绪原型:
  - 敬畏 (Awe)
  - 怀旧 (Nostalgia)
  - 审美愉悦 (Aesthetic Pleasure)
  - 道德义愤 (Moral Outrage)
  - 存在焦虑 (Existential Anxiety)

- 完善情绪 differentiation:
  - 增加时间维度评估 (持续时间)
  - 增加社会维度评估 (表达规则)
  - 增加功能维度评估 (适应性价值)

**理论依据**: Fehr & Russell (1984), Barrett (2017), SEP Emotion

---

### 2.2 预测加工 - 三传统整合优化

**优化内容**:
- 精细化预测误差计算:
  - 增加误差权重动态调整
  - 增加误差源交互效应建模
  - 增加跨时间误差追踪

- 主动推理策略扩展:
  - 增加"社会推理"策略 (通过他人验证预测)
  - 增加"叙事推理"策略 (通过故事重构预测)
  - 增加"时间推理"策略 (通过时间视角调整预测)

**理论依据**: Clark (2016), Friston (2010), Hohwy (2013)

---

### 2.3 集体情绪现象学增强

**优化内容**:
- 扩展集体情绪类型:
  - 增加"集体敬畏" (如群体观看日食)
  - 增加"集体怀旧" (如同学聚会)
  - 增加"集体希望" (如社会运动)

- 完善 Walther 四层评估:
  - 增加 Layer 5: 集体承诺 (联合行动的持续性)
  - 增加跨文化适配 (集体主义 vs 个人主义)

**理论依据**: Scheler (1954), Walther (1923), SEP Collective Intentionality
